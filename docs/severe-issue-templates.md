# 严重问题提示模板

## 核心原则

严重问题不能静默处理，也不能由系统擅自决定。

系统必须用简洁、明确、可选择的方式告诉用户问题，并给出 2-3 个处理方向。

## 通用模板

```text
我发现一个严重问题：【问题说明】。

这个问题会导致：【影响说明】。

你可以选择：
A. 【方案 A】
B. 【方案 B】
C. 【方案 C】

你确认后我再继续生成提示词。
```

## 冲突类模板

```text
我发现一个严重冲突：你同时要求【A】和【B】，这两个不能同时成立。

这个问题会导致生成结果方向混乱，模型可能无法稳定执行。

你可以选择：
A. 保留【A】
B. 保留【B】
C. 改成折中方案【C】

你确认后我再继续生成提示词。
```

## 参考素材冲突模板

```text
我发现一个参考素材冲突：【素材 1】中是【A】，但【素材 2】中是【B】。

由于【素材类型】属于必须锁定参考，我不能自动决定覆盖它。

你可以选择：
A. 保留【素材 1】的方向
B. 改用【素材 2】的方向
C. 重新上传一张统一后的参考图

你确认后我再继续生成提示词。
```

## 时长问题模板

```text
我发现这个分镜的时间不适合直接生成：【时长问题】。

这个问题会导致动作来不及完成，或单镜头稳定性下降。

你可以选择：
A. 延长到 6-15 秒范围内
B. 和相邻镜头合并
C. 拆成两个独立分镜

你确认后我再继续生成提示词。
```

## 素材不足模板

```text
这个部分目前缺少足够参考依据，我不建议直接硬写。

继续生成可能会导致风格不准、镜头语言空泛或生成结果不稳定。

你可以选择：
A. 上传参考图 / 分镜图 / 电影截图 / MV 截图
B. 让我通过互联网学习相关案例后再反推
C. 让我先用专业默认值生成一个通用版本

你确认后我再继续。
```

## 首尾帧无法衔接模板

```text
我发现首帧和尾帧之间无法作为同一条连续镜头自然衔接：【问题说明】。

这个问题会导致中间过程跳变，生成结果可能出现场景或人物漂移。

你可以选择：
A. 拆成两个镜头
B. 调整尾帧作为风格参考，不作为必须锁定尾帧
C. 重新提供可自然衔接的尾帧

你确认后我再继续生成提示词。
```

## 最终标准

严重问题要说明：问题是什么、为什么严重、用户可以怎么选。

不要只说“不行”。  
不要替用户决定核心方向。  
不要在严重冲突存在时继续生成最终提示词。
