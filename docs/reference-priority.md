# 参考素材优先级规则

## 核心原则

参考素材不能全部当成同等权重。系统必须说明每个素材参考什么，以及属于什么优先级。

固定三档：

1. 必须锁定参考
2. 强参考
3. 仅参考范围

## 必须锁定参考

以下素材永远属于必须锁定参考：

- 首帧图
- 尾帧图
- 人物模板图
- 角色定妆图
- 产品定型图
- 品牌主视觉定型图

必须锁定的内容包括身份、脸部特征、发型、服装、产品外形、标识位置、首尾帧构图、主体位置和核心空间关系。

## 强参考

常见强参考素材：

- 场景主参考图
- 舞台结构参考图
- 空间布局参考图
- 光影氛围参考图
- 分镜脚本图
- 镜头路径图
- 主色调参考图
- 核心风格参考图

强参考可电影化调整，但不能改变核心空间关系和主要视觉逻辑。

## 仅参考范围

常见仅参考范围素材：

- 材质参考图
- 表情参考图
- 局部细节图
- 色彩参考图
- 光效参考图
- 物件参考图

仅参考局部质感、色彩、表情方向或氛围范围，不能覆盖必须锁定参考。

## 冲突处理

如果参考素材冲突，系统不能自动决定，必须询问用户。

示例：人物模板是短发，但场景参考里人物是长发。人物模板属于必须锁定参考，不能自动改发型。

## 最终标准

每个参考素材都要写清：参考什么、锁定什么、可调整什么、不能覆盖什么。
