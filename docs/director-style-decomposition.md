# 导演风格拆解规则

## 核心原则

可以在提示词中写导演风格，但不能只写导演名字或风格标签。

导演风格必须转化成具体可执行的镜头语言、光线、空间、动作、节奏或情绪收束。

## 合格写法

```text
王家卫风格：长焦侧移、前景玻璃遮挡、霓虹散景、慢速人物动作、孤独情绪收束。
```

```text
Michael Bay 风格：低机位英雄仰拍、环绕推进、强逆光轮廓、背景爆破光浪、主体保持画面中心压迫感。
```

```text
希区柯克式心理压迫：缓慢推近、背景空间被长焦压缩、人物视线固定、环境声逐渐降低，最终用推拉变焦制造心理冲击。
```

## 不合格写法

```text
王家卫风格，高级电影感。
```

```text
希区柯克风格，很紧张。
```

```text
Michael Bay 风格，很燃很炸。
```

问题：只有风格标签，没有转化为镜头、动作、光线、空间或节奏。

## 使用规则

导演风格可以写在前面，但必须紧跟具体拆解。

推荐结构：

```text
【导演 / 风格标签】：【镜头语言】、【光线】、【空间关系】、【动作节奏】、【情绪收束】。
```

## 互联网学习规则

如果用户指定某位导演、某部电影、某支 MV、某个广告片或某种影像风格，而系统无法准确拆解，应使用互联网学习真实案例，或请用户提供截图、片段、链接、分镜图进行反推。

系统不能硬编导演风格。

## 最终标准

导演风格可以出现，但不能单独出现。  
必须转化成具体画面语言。  
越具体，模型越容易执行。
