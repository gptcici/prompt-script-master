# 专业摄影语言与空间关系规则

## 核心原则

AI 视频提示词必须用清楚的空间关系和镜头语言组织画面，不能只写抽象风格词。

## 空间关系

根据镜头目的选择空间描述，不机械要求每次都写前景、中景、背景、远景。

### 人物个体镜头

重点写人物位置、镜头距离、焦点、背景虚化和光线关系。

### 人物 + 宏观氛围

需要写人物与舞台、观众、灯光、远景之间的关系。

### 宏大场景

需要写空间层次、视觉中心、纵深、建筑或环境结构。

### 产品镜头

需要写产品、台面、反光、材质、背景光和品牌识别之间的关系。

## 构图术语

可以使用：

- 引导线构图
- 对称构图
- 中轴线构图
- 中心构图
- 框架式构图
- 负空间构图
- 纵深构图
- 长焦压缩构图
- 广角透视构图

构图术语不强制，但适合首帧、尾帧、ending pose、产品展示、人物英雄镜头、宏大场景建立镜头。

## 镜头语言

常用表达包括：

- 缓慢推近
- 低机位仰拍
- 高机位俯拍
- 横向跟拍
- 环绕推进
- 手持轻微晃动
- 长焦压缩背景
- 广角强化透视
- 推拉变焦制造心理冲击

## 焦点与景深

如果写焦点，必须明确：

1. 焦点在哪里。
2. 背景是否虚化。
3. 虚化程度如何。
4. 背景结构是否保留。

示例：

```text
焦点锁定人物眼睛和麦克风，背景时间环轻微虚化但保留圆形结构。
```

## 最终标准

空间关系要为画面目的服务。  
镜头语言要和动作、情绪、场景一致。  
构图术语可用但不能机械堆砌。
