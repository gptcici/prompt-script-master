# Output Format

## Seedance / 视频统一输出结构

默认采用 **中等精简 + 强锚点 + 镜头化时间轴**。所有最终视频提示词使用【正文提示词】+【负面提示词】；多镜头用明确时间段和镜头切点。旧模板已废弃。

```text
【正文提示词】
[时长]，[画幅]，[整体风格]。[一句话说明主体、场景、动作、是否多镜头]。

参考图职责：
[Image / Mixed 1] 仅用于锁定 [脸部/发型/服装/构图/建筑/色彩/动作]。
[Image / Mixed 2] 仅用于锁定 [...]。

全局光源锚点：
唯一主光源来自 [方向/高度/色温/软硬]，照亮 [人物/物体/环境的关键边缘]；暗部保持 [色彩和层次]；[云海/水面/墙面/地面] 只提供弱补光，不形成第二主光。所有阴影、轮廓光和高光方向统一。

全局风向 / 柔体动力学锚点：
[风力等级]的风从 [具体方向] 持续吹向 [具体方向]，风向全程统一；写清主体、道具、环境的分层受力。发丝/丝带/轻纱/厚布/纱幔/竹叶/云雾按不同材质表现速度差、重量差、延迟、回弹和重力下坠。正文只写正向物理目标，故障词放入负面提示词。

0-2秒，镜头一：[景别/机位/运镜]。[主体动作/表情/身体或材质联动]。[光影/风/景深/环境变化]。
2-4秒，镜头二：[同上，每镜头2-4句]
...

全局一致性：
[角色身份、服装、道具、建筑、场景、光源、风向、空间关系保持一致]。

【负面提示词】
人物类：...
动作类：...
风力类：...
场景类：...
风格类：...
```

## 人物镜头规则

模板中的每个有人物的镜头都必须先按《人物镜头强制覆盖规则》判断：人物核心动作和表情/肢体联动为必选；说话/唱歌/对口型时补足口型、下颌稳定、嘴唇开合和呼吸节奏；中近景/近景/特写时补足人物面部光影；存在发丝、衣物、饰品或环境动态材质时补足统一风向/动作惯性下的动态响应；有参考图时锁定人物身份。非触发模块不得硬写进最终提示词。

## 长度建议

15 秒多镜头视频：正文约 1200-1800 中文字，负面提示词约 120-300 中文字，通常 5-7 个镜头，每个镜头 2-4 句。涉及强风、发丝、布料、纱幔、云雾等复杂柔体动态时，可适度增加一个短风动力学模块，但避免重复解释。

如果平台没有独立负面提示词输入框，负面段不要混入主提示词。
