# 最终输出格式稳定规则

## 核心原则

最终输出必须稳定、清楚、可直接复制粘贴。

系统默认不展示内部评分、扣分过程、JSON、自检表或修改过程。

## 统一输出结构

单镜头和多镜头使用同一 2 段式结构（正文提示词 + 负面提示词），唯一区别是时间轴分段数量：单镜头 = 1 段，多镜头 = n 段。

> 最强格式规则：视频规格 + @Image1 参考图指令之后，必须立刻写人物核心外形特征。人物权重永远高于环境。其余结构不变。

> **旧版三段式（全程总定调/时间轴分段/全局收尾）已废弃，旧版 9 项时间轴格式已废弃。**

> 正文只写正向描述，不得出现"禁止""不要""避免""不得"类词汇。所有禁止项放入独立的负面提示词模块。

```text
【正文提示词】
视频规格 + 全程运镜总基调
@Image1 参考图标准化指令
人物核心外形特征
整体场景总设定

【时间轴分段】
景别 + 运镜状态 + 人物核心动作 + 衣发纱幔动态 + 人物光影 + 环境细节 + 景深变化

[权重标记] 整体风格质感 + 全局一致性约束

【负面提示词】
人物类：脸部变形、五官漂移、发型错乱、发饰丢失、服装变色
动作类：肢体错乱、手部畸形、手指粘连、动作僵硬
场景类：空间跳跃错位、纱幔无重力漂浮、道具错位
风格类：过度磨皮、过度CG感、云海平面贴图、卡通插画感、画面模糊低质
技术类：运镜抖动、景别突变、画幅错误
```

## 参考素材提醒

最终提示词输出后，必须提醒用户：

> 实际使用时，请加入对应参考素材，例如人物模板图、首帧 / 尾帧、分镜图、场景图、道具图或风格参考图，以提高一致性和生成稳定性。

如果提示词明显依赖参考素材，还要说明优先准备哪些素材。

## 输出简洁规则

最终输出应该是完整提示词，不是解释文章。

除非用户要求，否则不要展示：

- 评分
- 扣分项
- 思考过程
- 多轮修改记录
- 内部草稿

## 最终标准

用户拿到后可以直接复制到 AI 视频模型中使用。
