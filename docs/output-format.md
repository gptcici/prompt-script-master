# Output Format

## Seedance / 视频统一输出结构

默认采用 **中等精简 + 强锚点 + 镜头化时间轴**。所有最终视频提示词使用【正文提示词】+【负面提示词】；多镜头用明确时间段和镜头切点。旧模板已废弃。

```text
【正文提示词】
[时长]，[画幅]，[整体风格]。[一句话说明主体、场景、动作、是否多镜头]。

参考图职责：
[Image / Mixed 1] 仅用于锁定 [脸部/发型/服装/构图/建筑/色彩/动作]。
[Image / Mixed 2] 仅用于锁定 [...]。

全局光源锚点：
唯一主光源来自 [方向/高度/色温/软硬]，照亮 [人物/物体/环境的关键边缘]；暗部保持 [色彩和层次]；[云海/水面/墙面/地面] 只提供弱补光，不形成第二主光。所有阴影、轮廓光和高光方向统一。

全局物理动力学锚点：
[主要动力来源：风 / 人物动作 / 道具接触 / 雨雪 / 水流 / 烟雾 / 火焰 / 粒子 / 重力 / 惯性 / 机械运动]。[方向/力量/节奏] 全程保持一致或按剧情有明确变化。写清发丝、衣料、饰品、纱幔、云雾、水面、雨线、烟尘、火焰、粒子、道具、接触点、碰撞、重力下坠、惯性延迟、速度差、反作用、最终停点或消散状态。简单风动可写成“全局风向 / 柔体动力学锚点”，复杂动态必须使用本模块。


0-2秒，镜头一：[景别/机位/运镜]。[主体动作/表情/身体或材质联动]。[光影/风/景深/环境变化]。
2-4秒，镜头二：[同上，每镜头2-4句]
...

全局一致性：
[角色身份、服装、道具、建筑、场景、光源、风向、空间关系保持一致]。

【负面提示词】
人物类：...
动作类：...
风力类 / 动力学类：...
场景类：...
风格类：...
```


## 人物镜头规则

模板中的每个有人物的镜头都必须先按《人物镜头强制覆盖规则》判断：人物核心动作和表情/肢体联动为必选；说话/唱歌/对口型时补足口型、下颌稳定、嘴唇开合和呼吸节奏；中近景/近景/特写时补足人物面部光影；存在发丝、衣物、饰品、环境动态材质、流体、粒子、道具接触、碰撞、重力或惯性时补足统一物理动力学响应；有参考图时锁定人物身份。非触发模块不得硬写进最终提示词。

## 长度建议

输出长度按 `references/output-length-mode-standard.md` 选择：压缩版约 80-180 中文字；快速视频版约 120-260 中文字；标准视频版约 500-1000 中文字；高控制视频版约 1000-1600 中文字；密集分镜 / shot list / storyboard 仅在用户明确要求时超过 1600 字。15 秒视频默认 2-4 个镜头，锁脸、唱歌、口型、多人物或复杂动力学优先 2-3 个镜头，不再默认硬塞 5-7 个镜头。负面提示词约 120-300 中文字。

如果平台没有独立负面提示词输入框，负面段不要混入主提示词。

## V0.9.76 镜头意图增强

复杂镜头、剧情镜头、人物动作镜头、多人镜头和强光影镜头，建议在内部先判断：这个镜头的唯一意图是什么。最终提示词中不必写成表格，但每个关键镜头应自然包含：镜头目的、人物/物体动作落点、多人主次、光源动机、材质反射、起始状态和结束状态。

可替换镜头句式：

```text
0-2秒，镜头一：[景别/机位/运镜]，镜头意图是 [建立/揭示/情绪转折/材质证明]。[主体动作 + 物理结果 + 动作终点]。[光源方向/色温/阴影/反射/风动/景深/环境变化]。
```


## 模式优先提示

图生视频、首尾帧、视频续写和图片生成不要直接套用完整 T2V 长模板。I2V 只写图片看不出来的动作、镜头、光线变化、动力学和保持约束；IMAGE 默认自然语言段落，不输出视频时间轴。


## V0.9.81 参考图可识别性与人脸完整性

图生视频、首尾帧、参考图锁脸和图片重绘任务在执行 I2V 极简原则前，必须先确认参考图中的人脸、服装、背景、构图是否清晰可识别。清晰部分可以不重复描述；不清晰部分必须补写；人脸若存在畸形、异常色块、黑斑、异常阴影、大面积死黑或不符合生理逻辑的皮肤质感，不得作为身份细节继承，必须用自然面部结构、肤色连续性和主光源面部阴影逻辑重建，并把故障项放入【负面提示词】人物类。


## V0.9.82 参考图上传后自检

当任务包含参考图、首帧、尾帧、人物图、服装图、场景图、构图图、图生视频或图片重绘时，最终提示词生成前必须先执行参考图上传后自检：判断关键区域是否清晰、人脸是否符合生理逻辑、哪些内容可继承、哪些内容必须补写、是否需要建议用户重新生成参考图。只有参考图关键视觉信息清晰时，才执行“只写图片看不出来的东西”的 I2V 极简原则。
