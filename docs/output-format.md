# 最终输出格式稳定规则

## 核心原则

最终输出必须稳定、清楚、可直接复制粘贴。

系统默认不展示内部评分、扣分过程、JSON、自检表或修改过程。

## 单镜头默认结构

单镜头提示词默认使用：

```text
【生成规格】
【参考素材说明】
【镜头目标】
【时间轴】
【镜头语言与摄影参数】
【灯光、场景与动态】
【一致性要求】
【禁止项】
```

## 多镜头默认结构

多镜头提示词默认使用：

```text
【项目总设定】
【参考素材优先级】
【分镜 1】
【分镜 2】
【分镜 3】
【镜头之间的连续性要求】
【统一一致性要求】
【统一禁止项】
```

## 参考素材提醒

最终提示词输出后，必须提醒用户：

> 实际使用时，请加入对应参考素材，例如人物模板图、首帧 / 尾帧、分镜图、场景图、道具图或风格参考图，以提高一致性和生成稳定性。

如果提示词明显依赖参考素材，还要说明优先准备哪些素材。

## 输出简洁规则

最终输出应该是完整提示词，不是解释文章。

除非用户要求，否则不要展示：

- 评分
- 扣分项
- 思考过程
- 多轮修改记录
- 内部草稿

## 最终标准

用户拿到后可以直接复制到 AI 视频模型中使用。
