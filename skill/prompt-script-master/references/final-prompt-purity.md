# 最终提示词纯净度

最终提示词只给 AI 视频生成模型看，只能包含模型需要执行的内容。

> **边界说明**：本规则针对的是最终交付给视频生成模型的提示词成品。时间轴分段方案、确认沟通、方案建议等中间工作产物不适用本规则——这些是面向用户而非面向模型的。中间产物经用户确认后，才提炼为符合纯净度的最终提示词。

## 禁止进入最终提示词的内容

- 内容理解与专业复述
- 参考资料识别与缺失资料建议
- 动作 / 舞蹈 / 表演方案预览
- 经典镜头与高潮运镜建议
- 选择项、解释、询问、确认清单
- “我建议”“你可以选择”“是否采用”“请确认”等对话性语言
- 动作库组合名称
- 经典镜头库名称

## 正确做法

用户确认后，把被采用的信息翻译成可执行画面语言。

错误：采用迈克尔·贝式低角度环绕英雄镜头。

正确：摄影机置于低于人物胸口的位置，使用 28mm 广角从左前方中景开始，以弧线轨迹向正面半环绕并同步推近到半身近景，焦点锁定人物眼神，背后强光和烟雾形成运动视差。

错误：采用副歌爆发开场动作。

正确：角色低头站定，副歌强拍前半秒抬眼看向镜头，右脚向前迈一步，右手从胸前向外打开，最后在半身近景中完成有力量的定格。
