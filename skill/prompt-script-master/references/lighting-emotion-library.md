# 光影情绪库

用于把音乐、剧情、舞台和人物情绪转译成可拍摄的光线方案。适用于 MV、演唱会、剧情短片、人物情绪和舞台项目。

## 基础映射

| 光影 | 情绪功能 | 常见用途 |
|---|---|---|
| 单束追光 + 暗背景 | 孤独、被注视、内心独白、舞台仪式 | 开场、独唱、钢琴、慢歌 |
| 冷白 / 微蓝光 | 回忆、空灵、距离感、脆弱 | 抒情、失恋、梦境、深夜 |
| 暖金 / 琥珀光 | 怀旧、亲密、浪漫、温柔 | 日落、民谣、回忆、观众对话 |
| 深蓝 / 青蓝 wash | 宇宙感、海洋感、忧郁、沉浸 | 梦幻段、慢歌、视觉空间延展 |
| 粉色 / 玫红柔光 | 甜美、亲和、少女感、爱情明亮面 | 可爱段、轻快副歌、互动 |
| 正红 / 暗红 | 戏剧、力量、愤怒、传统仪式感 | 高潮、决裂、东方叙事、强情绪 |
| 紫 / 蓝紫 | 华丽、神秘、梦幻、舞台奇观 | 开场、换装、高潮前 |
| 火光 / 橙红跳动光 | 愤怒、毁灭、激情、关系断裂 | 情绪崩塌、决裂、工业空间 |
| 明亮自然光 | 轻松、喜剧、日常、希望 | 白天外景、约会、群像 |
| 日落逆光 | 理想化、自由、回忆、浪漫但短暂 | 海边、天台、道路、离别 |
| 硬光高对比 | 力量、性感、粗粝、夸张喜剧 | 快节奏、身体展示、夏日外景 |
| 烟雾 + 光束 | 空间纵深、舞台规模、梦境、高潮 | 演唱会、舞台、Live MV |

## 光型功能

- 追光：强制锁定主体，适合主唱、独白、登场、ending pose。
- 逆光：勾勒轮廓，让头发、服装边缘、烟雾和身体姿态形成记忆点。
- 侧光：增强面部立体和情绪矛盾，适合近景和中近景。
- 顶光：制造仪式感、压迫感或舞台戏剧性。
- 彩色 wash：统一空间情绪，适合舞台或梦境空间。
- 实用光源：烛光、台灯、火光、屏幕光、车灯、窗光，用于提升真实感。

## 情绪升级方式

### 抒情段

暗背景 + 单束追光 / 柔侧光 → 面部特写 → 光线轻微减弱或拉远收束。

### 预副歌

主光保持 → 背景光逐渐增强 → 彩色 wash 加入 → 镜头节奏开始加快。

### 副歌高潮

逆光、光束、烟雾、全场 wash 同时释放，但每个镜头最多强调 1-2 个主光效，避免堆叠。

### 喜剧 / 轻快段

明亮自然光或高饱和彩色光，动作和光线都更直接，不需要过度电影化阴影。

### 决裂 / 愤怒段

火光、橙红、硬阴影、烟雾或工业冷光形成冲突。动作可更夸张，但要保持可拍摄。

## 输出约束

最终提示词中不要只写“高级灯光”“电影感灯光”。必须写清：

- 光源来自哪里。
- 颜色或色温是什么。
- 照亮主体哪一侧。
- 背景和烟雾如何参与画面。
- 灯光如何随音乐或剧情变化。
