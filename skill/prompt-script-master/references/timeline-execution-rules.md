# Timeline Execution Rules

## 核心原则

时间轴是视频提示词的执行主体。每个时间段都是黄金语序的**分段适配版**：首段建立角色主体和外形特征，后续段落不重复角色描述，专注于本段的变化信息。

> **来源**: 基于 `seedance-prompt-order-rules.md` 黄金语序（SKILL.md §黄金语序）的分段适配

## 分段触发条件

当视频时长超过 5 秒，或用户说明时长大于 X 秒，且用户没有明确要求"一镜到底"时：

1. **询问用户**："视频时长预计超过 5 秒，是否需要分段？"
2. **不定长**：分段没有固定时长。根据每段的内容权重合理分配时间——动作密集、空间揭示复杂的段落多给时间，简单转场少给。
3. **自查**：分配完时长后，先检查一遍：
   - 各段时间分配是否匹配该段的动作密度？
   - 总时长是否等于目标视频长度？
   - 段间推进逻辑是否流畅？
4. **确认**：将分段方案和时长分配展示给用户确认，确认后再生成时间轴内容。

当用户明确要求"一镜到底"时，不询问分段，直接按单镜头生成完整时间轴内容。

## 时间轴分段格式

每个时间段的字段格式：

```text
本段景别 + 本段运镜状态 + 焦点变化 + 核心动作 + 表情肢体联动 + 人物面光/骨骼光影 + 环境光影变化 + 环境细节补充 + 本段景深变化
```

各字段说明：

| 字段 | 说明 |
|---|---|
| 本段景别 | 远景/全景/中景/近景/特写，可写景别变化 |
| 本段运镜状态 | 固定/推/拉/摇/跟/环绕/升降，含方向和速度 |
| 焦点变化 | 本段焦点对象和切换逻辑 |
| 核心动作 | 主体正在做什么 |
| 表情肢体联动 | 面部表情 + 头颈肩手/身体重心的联动变化 |
| 人物面光/骨骼光影 | 主体主光方向、骨骼锚点明暗、眼神光 |
| 环境光影变化 | 场景光源变化、阴影、色温、亮度 |
| 环境细节补充 | 背景、道具、空间层次的具体变化 |
| 本段景深变化 | 前后景虚实切换、景深推进或释放 |

## 首段与后续段落

- **首段** (0-xs)：除以上字段外，必须建立人物主体 + 关键外形特征（发型/服装/体型/肤色等），后续段落不再重复。
- **后续段落**：仅写本段变化，保持角色连续性和运镜衔接，不重复角色外貌。

## 板块规则

- `整体目标与风格基调` 说明镜头目标、整体风格、情绪基调和主视觉方向。
- `时间轴` 承载所有具体执行信息，每个时间轴段落按上述格式逐项写入。
- `全局摄影基调` 说明画幅、色彩、质感和总体摄影风格。
- 不要单独设置 `灯光、场景与动态` 或 `镜头语言与摄影参数` 作为主执行板块——这些信息必须进入每个时间轴段落。

## 时间轴段落示例

（以下为例示格式，实际时长由分段确认流程决定）

```text
首段：
本段景别：远景
本段运镜状态：固定机位
焦点变化：焦点锁定站立窗口的女孩
核心动作：女孩站在窗边，抬手触碰窗帘
表情肢体联动：眼神安静望向窗外，嘴角微抿，肩膀轻微耸起，身体重心略前倾
人物面光/骨骼光影：晨光从窗缝斜入，照亮右侧颧骨和鼻梁，左颊处于柔暗，眼神光微弱
环境光影变化：室内整体偏暗，窗缝光在地面形成细长亮带
环境细节补充：旧木地板反光微弱，窗台上干花束剪影清晰
本段景深变化：浅景深，焦点在女孩脸的侧影，窗帘纹理微虚

中段：
本段景别：中景 → 近景（慢推）
本段运镜状态：缓慢推近，机位从正前方微升
焦点变化：焦点从脸转移到瞳孔
核心动作：女孩拉开窗帘，光线涌入
表情肢体联动：眉头微松，眼睑轻抬适应光线，头部微微后仰，手指缓慢松开帘布
人物面光/骨骼光影：正面强光涌入，鼻梁高光面积扩大，下颌明暗交界线下移，眼神光增强为两点
环境光影变化：室内暗部快速退向角落，窗框投影在地面拉长旋转
环境细节补充：空中微尘被阳光照亮成悬浮光点
本段景深变化：浅景深持续，背景从暗部逐渐显露出窗外树影轮廓

末段：
本段景别：近景微仰
本段运镜状态：微仰固定，轻微呼吸感手持微晃
焦点变化：焦点稳定在瞳孔，背景继续保持浅景深虚化
核心动作：女孩眯眼看向窗外，瞳孔收缩
表情肢体联动：嘴角微微上扬，眼尾轻微皱起，呼吸平缓，肩膀放松
人物面光/骨骼光影：正面自然光均匀照亮面部，眼眶下方保留微弱阴影，皮肤质感细腻
环境光影变化：窗外树影斑驳洒在地板和墙面，光斑轻微摇曳
环境细节补充：窗外可见初秋枝叶和远处淡蓝天际线
本段景深变化：焦点仍锁定面部，窗外景物保持浅景深虚化
```

## 不合格信号

以下情况视为不合格：

- 时间轴只有动作描述，缺少景别/运镜/焦点/光线/景深任一维度。
- 分段之间动作跳跃不连续（穿帮）。
- 后续段落重复首段已建立的角色外貌描述。
- 光线、场景、镜头信息只出现在独立板块中，没有进入每个时间轴段落。
- 段落开头没有说明与前一段的关系。
