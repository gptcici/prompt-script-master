# Timeline Execution Rules

## 核心原则

时间轴是视频提示词的执行主体。每个时间段都要写成完整的视频生成指令，而不是动作摘要。

## 每个时间段必须包含

每一段时间轴都必须同时写入：

1. 衔接关系：开场建立、衔接上一段、镜头切到、从上一段继续、由上一段转入。
2. 画面动作：主体正在做什么。
3. 场景动态：背景、道具、空间层次如何变化。
4. 光线变化：主光、轮廓光、反光、阴影、色温或亮度如何变化。
5. 镜头控制：推近、拉远、横移、跟随、环绕、升降、变焦、切焦或保持静止。
6. 摄影信息：摄影机视角、焦段或镜头类型、景别变化、构图、焦点对象、景深状态。
7. 情绪节奏：这一段的情绪推进、爆发或收束。

## 板块规则

`整体目标与风格基调` 用于说明镜头目标、整体风格、情绪基调和主视觉方向。

`时间轴` 用于承载所有具体执行信息。

`全局摄影基调` 只用于说明整体一致的画幅、色彩、质感和总体摄影风格。

不要单独设置 `灯光、场景与动态` 作为主执行板块。场景、光线、动态都必须进入每个时间轴段落。

不要单独设置 `镜头语言与摄影参数` 作为主执行板块。镜头运动、焦段、焦点、景别、构图都必须进入每个时间轴段落。

## 时间轴段落格式

```text
0-3 秒：开场建立 / 衔接上一段，【主体动作】，【场景动态】，【光线变化】，【摄影机 + 焦段 / 镜头类型 + 运镜方式】，【景别变化 + 构图】，【焦点对象 + 景深状态】，【情绪节奏】。
```

## 合格示例

```text
0-4 秒：开场建立舞台空间，女歌手站在舞台中央抬眼唱出副歌，背后观众灯海低亮度闪烁，正面柔光照亮面部，摄影机使用 85mm 长焦从正前方中景缓慢推近到半身近景，焦点锁定她的眼神和麦克风，背景时间环保持浅景深虚化，情绪从克制进入爆发前的蓄力。
4-8 秒：衔接上一段推近动作，副歌强拍落下，背后时间环逐层点亮，舞台轮廓光穿过烟雾，摄影机继续轻微推近并略微仰拍，景别收束到胸像近景，焦点稳定跟随歌手面部，观众灯海向两侧延展，情绪完成英雄感爆发。
```

## 不合格信号

以下情况视为不合格：

- 时间轴只写主体动作，没有镜头控制。
- 时间轴只写场景变化，没有光线变化。
- 时间轴只写运镜，没有焦段、景别或焦点。
- 具体光线、场景、焦点、焦段只出现在单独板块中，没有进入时间轴。
- 时间段开头没有说明与前后段落的关系。
