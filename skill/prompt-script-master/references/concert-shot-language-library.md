# 演唱会镜头语言库

用于舞台、Live MV、演唱会和大型表演类脚本。目标是让时间轴同时具备人物情绪、舞台空间和现场观众反馈。

## 景别功能

### 大全景 / 舞台全景

用途：建立场馆规模、舞美结构、LED 屏、灯光阵列、观众灯海和人物所在位置。

适用位置：开场、段落切换、副歌爆点、结尾收束。

写法要求：必须交代人物在舞台中的位置，例如中央、升降台、T 台前端、主舞台边缘、观众环绕处。

### 中景 / 中近景

用途：承载歌手动作、握麦、站姿、服装轮廓、身体节奏和表演状态。

适用位置：主歌、预副歌、情绪推进、对口型段落。

写法要求：必须写清上半身动作、手势、视线方向和麦克风状态。

### 特写 / 大特写

用途：捕捉眼神、嘴唇对口型、汗水、发丝边缘光、手握麦克风、观众表情或乐器细节。

适用位置：歌词情绪转折、强拍前一瞬、高潮前的蓄力、结尾记忆点。

写法要求：特写不能连续过多，必须与中景或全景呼吸切换。

## 运镜功能

- 固定机位：用于抒情、独白、情绪锚点和正式舞台感。
- 缓慢推近：用于逐步进入人物内心或强化主唱气场。
- 缓慢拉远：用于孤独感、章节收束或将个人情绪还给全场。
- 横移 / 轨道：用于展示服装流动、舞台层次、人物缓步走位。
- 摇臂：用于从人物打开到舞台规模，或从观众扫向主唱。
- 手持 / 稳定器：少量用于临场感、观众区、舞台边缘互动和高能走位。
- 环绕：只在情绪顶点或双人关系段落少量使用，避免滥用。

## 三线交替

最终时间轴必须交替调度：

1. 主体线：歌手动作、面部、握麦、对口型、身体情绪。
2. 空间线：舞台、灯光、LED、烟雾、T 台、场馆纵深。
3. 观众线：合唱、挥手、灯海、手机屏、面部反应、欢呼。

如果用户要求“一个歌手演唱会镜头”，不要只写歌手。至少在时间轴中加入一次空间线和一次观众线。

## 镜头组合公式

### 开场仪式

舞台全黑或低照度大全景 → 追光落下 → 中景锁定主唱 → 面部或手部特写。

### 抒情慢歌

稳定中景 → 面部特写 → 细节特写 → 中景缓慢拉远。

### 预副歌推进

中景走位 → 轨道轻推 → 观众或乐手切镜 → 主唱回到画面中心。

### 副歌高潮

主唱特写 → 观众反应 → 舞台全景灯光释放 → 逆光英雄姿态。

### 结尾收束

主唱完成 ending pose → 观众合唱或灯海 → 高位拉远 → 黑场或静止记忆点。

## 禁止项

- 不要连续多个段落都使用同一景别。
- 不要把“燃炸”“电影感”“很震撼”当成镜头语言。
- 不要堆叠过多运镜，例如同一段同时航拍、推拉、变焦、旋转、摇臂。
- 不要在最终提示词中出现本库的公式名或镜头库名。
