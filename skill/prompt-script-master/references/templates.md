# Skill Templates

## 单镜头输出结构

```text
【生成规格】
生成【时长】秒，【画幅】，【模型 / 模式】，整体风格为【风格】。

【参考素材说明】
【素材名称】作为【参考类型】，属于【优先级】，主要参考【用途】。

【镜头目标】
本镜头主要表达【主体】在【场景】中的【动作 / 状态 / 情绪 / 目的】。

【时间轴】
0-【时间点】秒：【主体动作 + 环境变化 + 镜头控制 + 景别变化 + 焦点控制】。
【时间点】-【结束】秒：【主体动作 / ending pose + 环境变化 + 镜头控制 + 景别变化 + 焦点控制】。

【镜头语言与摄影参数】
全局使用【摄影机 / 镜头】，焦段【焦段】，整体景深【景深基调】。注意：这里只写全局摄影基调，具体运镜必须已经写入每个时间轴段落。

【灯光、场景与动态】
【光线方向】，【场景动态】，【空间层次】。

【一致性要求】
保持【人物 / 产品 / 场景 / 服装 / 道具】一致。

【禁止项】
不要【异常1】，不要【异常2】，不要字幕、水印、乱码文字。
```

## 时间轴段落模板

每个时间段都必须使用以下思路：

```text
【起止时间】：【主体正在做什么】，【场景 / 光线 / 道具如何变化】，【摄影机如何运动】，【景别从什么变到什么】，【焦点锁定或转移到哪里】。
```

示例：

```text
0-4 秒：女歌手站在舞台中央抬眼唱出副歌，背后观众灯海轻微闪烁，摄影机从正前方中景缓慢推近到半身近景，焦点锁定她的眼神和麦克风，背景时间环保持柔和虚化。
4-8 秒：副歌强拍落下，时间环逐层点亮，摄影机继续轻微推近并略微仰拍，景别收束到胸像近景，焦点稳定跟随歌手面部，舞台逆光穿过烟雾形成轮廓光。
```

## 禁止的时间轴写法

不要这样写：

```text
【时间轴】
0-4 秒：女歌手站在舞台中央唱歌。
4-8 秒：时间环点亮。

【镜头语言】
镜头缓慢推近，焦点锁定女歌手。
```

原因：模型无法知道哪个时间段该怎样运镜。

## 参考素材说明

```text
【参考素材清单】
1. 【素材名称】：【素材类型】
   - 优先级：【必须锁定参考 / 强参考 / 仅参考范围】
   - 主要用途：【用途】
   - 必须保持：【不可改变内容】
   - 可以调整：【可调整内容】
```

## 负面提示词原则

禁止项只写关键异常。一般 5-10 条，复杂项目最多约 12 条。
