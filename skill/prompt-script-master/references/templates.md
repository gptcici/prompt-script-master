# Skill Templates

## 单镜头输出结构

```text
【生成规格】
生成【时长】秒，【画幅】，【模型 / 模式】，整体风格为【风格】。

【参考素材说明】
【素材名称】作为【参考类型】，属于【优先级】，主要参考【用途】。

【镜头目标】
本镜头主要表达【主体】在【场景】中的【动作 / 状态 / 情绪 / 目的】。

【时间轴】
0-【时间点】秒：【主体动作 + 镜头变化 + 环境变化】。
【时间点】-【结束】秒：【ending pose / final frame / 情绪收束】。

【镜头语言与摄影参数】
使用【摄影机 / 镜头】，焦段【焦段】，镜头从【起点】向【终点】进行【运镜方式】，焦点锁定【焦点对象】。

【灯光、场景与动态】
【光线方向】，【场景动态】，【空间层次】。

【一致性要求】
保持【人物 / 产品 / 场景 / 服装 / 道具】一致。

【禁止项】
不要【异常1】，不要【异常2】，不要字幕、水印、乱码文字。
```

## 参考素材说明

```text
【参考素材清单】
1. 【素材名称】：【素材类型】
   - 优先级：【必须锁定参考 / 强参考 / 仅参考范围】
   - 主要用途：【用途】
   - 必须保持：【不可改变内容】
   - 可以调整：【可调整内容】
```

## 负面提示词原则

禁止项只写关键异常。一般 5-10 条，复杂项目最多约 12 条。
