# Seedance 情绪表情与肢体联动动作库

适配版本：Seedance 2.0 / 2.1

## 设计标准

本库用于把抽象情绪翻译成 Seedance 更容易识别的可见表演动作。每个模块必须同时包含：

```text
整体情绪基调 + 眼部动作 + 眉部动作 + 嘴部动作 + 面部肌肉状态 + 动态变化节奏 + 头颈肩手/身体联动
```

所有模块均为正向具象描述，禁止在正文里写“不要面瘫、不要崩脸、不要僵硬”等否定句。负面词只放入负面提示词栏。

## 调用规则

- 单镜头只选 1 个核心情绪模块。
- 长镜头需要多情绪时，按时间轴拆成起始、过渡、结尾。
- 特写优先微表情；中景使用标准版；远景/全景只保留身体联动。
- 真实人物动作幅度收窄，卡通/玩偶动作可放大，古风人物节奏放缓并增加袖摆、裙摆、发丝联动。
- 表情模块固定放在“主体 + 核心动作”之后，光影和场景之前。

## 正向情绪模块 E001-E010

### E001 温柔浅笑
- 轻微：眼尾轻轻弯起，嘴角小幅上扬，脸颊肌肉自然放松，头部微微侧倾，肩颈松弛。
- 标准：眼神柔和注视前方，眉心舒展，嘴角逐渐扬起成温柔浅笑，呼吸平稳，手部动作缓慢。
- 强化：笑意从眼底缓缓扩散到嘴角，苹果肌轻微抬起，头部轻轻侧转，发丝和衣袖随动作轻微摆动。

### E002 开怀大笑
- 轻微：眼睛微眯，嘴角明显上扬，肩膀轻轻起伏。
- 标准：眼尾弯起，嘴唇自然张开露出笑意，脸颊上提，身体微微前倾。
- 强化：笑意完全展开，眉眼明亮，肩膀和胸腔随笑声起伏，手部自然抬起配合情绪。

### E003 腼腆害羞
- 轻微：眼神短暂躲闪，嘴角轻轻抿起，头部小幅低下。
- 标准：眼睑微垂，眉尾放软，嘴角含笑，脸颊微微收紧，双手靠近身体。
- 强化：视线从对方脸上移开，头部缓缓低下，肩膀轻收，手指轻捏衣袖或发梢。

### E004 松弛治愈
- 轻微：眼神放松，嘴唇微微张合，呼吸节奏平缓。
- 标准：眉眼舒展，嘴角带很浅的笑意，肩膀自然下沉，身体重心稳定。
- 强化：整个人从紧绷逐渐放松，眼神变柔，胸口呼吸起伏明显变慢，手臂自然垂落。

### E005 俏皮灵动
- 轻微：眼神快速一闪，眉梢微挑，嘴角带一点狡黠笑意。
- 标准：眼睛明亮，眉尾上扬，嘴角偏向一侧，头部轻快侧转。
- 强化：眨眼后露出俏皮笑意，肩膀轻轻一抬，手部做小幅摆动或点指动作。

### E006 宠溺包容
- 轻微：眼神柔和下垂，嘴角轻弯，呼吸平稳。
- 标准：眼睛专注看向对方，眉眼舒展，嘴角温和上扬，手部缓缓靠近。
- 强化：笑意缓慢浮现，头部微微低下靠近对方，肩颈放松，手掌轻柔停顿。

### E007 惊喜雀跃
- 轻微：眼睛微微睁大，眉毛上抬，嘴角露出笑意。
- 标准：瞳孔方向快速聚焦，眉梢抬起，嘴唇轻启，身体微微前倾。
- 强化：眼睛明显睁大，笑容迅速展开，肩膀轻跳，双手自然抬起。

### E008 安心释然
- 轻微：眉心逐渐舒展，嘴角放松，呼吸变慢。
- 标准：眼神从紧张变柔，嘴唇轻轻吐气，下颌放松，肩膀下沉。
- 强化：人物长长舒出一口气，眼尾变柔，身体重心稳定下来，双手松开。

### E009 憧憬向往
- 轻微：视线抬向远方，眼神明亮，嘴角微弯。
- 标准：眉眼舒展，眼睛看向高处或远处，嘴唇轻轻张开，身体略向前。
- 强化：眼神逐渐发亮，头部抬起，胸口轻微起伏，手部向前或向上伸出。

### E010 傲娇得意
- 轻微：眉梢轻挑，嘴角单侧上扬，眼神短暂移开。
- 标准：下巴微抬，眼神斜看，嘴角含笑，肩膀轻轻后收。
- 强化：人物轻哼式表情，下巴上扬，手臂轻抱或袖摆一甩，笑意克制但明显。

## 负向情绪模块 E101-E110

### E101 悲伤低落
- 轻微：眼神下垂，眉尾放低，嘴角松弛下压。
- 标准：眼眶微红，眉心轻蹙，嘴唇抿紧，头部缓慢低下。
- 强化：眼神失焦，肩膀下沉，呼吸变浅，手指无力收紧。

### E102 隐忍落泪
- 轻微：眼眶泛湿，嘴唇轻抿，下颌微微绷紧。
- 标准：泪光聚在眼角，眉心轻皱，嘴角努力保持平稳，肩膀轻颤。
- 强化：泪珠从眼角滑落，下颌控制不住轻颤，头部微低，手指攥紧衣袖。

### E103 崩溃痛哭
- 轻微：眼神破碎，嘴唇颤动，呼吸急促。
- 标准：眉心紧皱，嘴巴自然张开，眼泪滑落，肩膀明显颤抖。
- 强化：面部肌肉失控收紧，哭泣带动胸腔起伏，身体前倾，双手捂脸或扶住身体。

### E104 愠怒生气
- 轻微：眉头压低，眼神变冷，嘴角收紧。
- 标准：眉心紧蹙，眼睛直视目标，嘴唇抿成一线，下颌绷紧。
- 强化：眼神凌厉，鼻翼轻动，下颌线紧绷，肩膀上提，手掌握拳。

### E105 隐忍愤恨
- 轻微：眼神压低，嘴唇紧闭，呼吸压抑。
- 标准：眉头缓慢收紧，眼神含怒但克制，手指慢慢握紧。
- 强化：怒意从眼底逐渐累积，下颌绷紧，肩颈僵住，身体重心前压。

### E106 焦虑不安
- 轻微：眼神左右游移，眉心轻蹙，嘴唇微抿。
- 标准：眨眼频率略快，呼吸变浅，肩膀轻收，手指小幅摩挲。
- 强化：视线不断寻找出口，嘴唇轻颤，肩颈紧绷，双手不自觉攥紧。

### E107 恐惧害怕
- 轻微：眼睛微微睁大，眉毛上抬，嘴唇轻启。
- 标准：瞳孔方向锁定危险来源，眉眼紧张，下颌后缩，身体微微后退。
- 强化：眼睛明显睁大，嘴唇张开，肩膀抬起，手臂本能护在身前。

### E108 委屈难过
- 轻微：眼尾下垂，嘴角轻压，视线避开。
- 标准：眼眶泛红，眉尾向内收，嘴唇微微抿住，头部低下。
- 强化：泪光在眼中打转，嘴唇轻颤，肩膀轻缩，手指紧紧捏住衣角。

### E109 愧疚自责
- 轻微：眼神下移，眉心轻皱，嘴角失去弧度。
- 标准：人物不敢直视前方，头部缓慢低下，双手收在身前。
- 强化：眉眼沉重，嘴唇紧闭，肩膀内扣，身体重心后撤。

### E110 落寞孤寂
- 轻微：眼神放空，嘴角松弛，身体静止。
- 标准：视线看向远处但不聚焦，眉眼低垂，肩膀自然塌下。
- 强化：人物静静站立，呼吸很轻，手臂无力垂落，背影与环境距离感增强。

## 中性情绪模块 E201-E208

### E201 平静淡然
轻微眨眼，眉眼平稳，嘴唇自然闭合，肩颈放松，身体重心稳定。

### E202 专注认真
眼神锁定目标，眉心微收，嘴唇轻抿，头部略向前，手部动作稳定。

### E203 惊讶诧异
眼睛睁大，眉毛上抬，嘴唇轻启，头部轻微后仰，身体动作短暂停顿。

### E204 思索沉吟
视线微微下移，眉心缓慢收紧，嘴唇轻抿，手指靠近下巴或轻触物件。

### E205 疑惑不解
眉头一侧微挑，眼神短暂偏移，嘴唇微张，头部小幅侧倾。

### E206 笃定坚定
眼神逐渐聚焦，眉心舒展但有力量，嘴唇闭合稳定，头部抬起，肩膀打开。

### E207 严肃凝重
眉眼压低，嘴角收紧，面部肌肉克制，肩颈稳定，身体保持静止。

### E208 慵懒倦怠
眼皮微垂，嘴唇放松，头部轻偏，肩膀下沉，动作节奏变慢。

## 特殊动态模块 E301-E306

### E301 说话 / 演唱口型
嘴唇按发声节奏自然开合，下颌稳定，呼吸带动胸口轻微起伏，眼神与情绪保持一致。

### E302 自然眨眼
眼睑小幅闭合再睁开，动作平滑不突兀，眨眼后眼神方向保持连续。

### E303 回眸转头
头部先转，眼神随后跟上，肩颈与发丝同步带动，表情在转头过程中自然变化。

### E304 呼吸起伏
胸口和肩膀随呼吸轻微起伏，紧张时呼吸浅快，放松时呼吸缓慢平稳。

### E305 行走 / 奔跑表情联动
脚步节奏影响肩膀、发丝、衣摆和面部稳定度，表情不能像静态贴图一样固定。

### E306 行礼 / 致意动作联动
头部低下、肩膀收束、手部动作与眼神变化同步，古风角色需加入衣袖和裙摆轻微跟随。
