# 复述阶段流程

本文件定义确认前复述阶段。只要用户提出 AI 视频提示词、分镜、MV、广告、舞台、首尾帧、图生视频或提示词优化需求，就必须先进入复述阶段。

## 最高原则

复述阶段是用户沟通层，不是最终提示词阶段。用户未完整确认前，不得输出最终提示词、完整时间轴或可直接复制给视频模型的生成指令。

## 必定触发环节

每次复述阶段必须包含：参考资料识别、内容理解复述、项目类型判断、信息完整度判断、动态级别建议、2-3个问题、确认提醒。

## 快捷生成例外

如果用户明确说“直接生成/不用问/你决定/跳过确认/按默认来”，可跳过显性确认流程，但不得在最终提示词中输出解释、选项或确认语。

## 强制中断条件

以下情况必须停止并询问：核心主体不清、多主体冲突、参考冲突、无法闭合时间轴。

## 约束

未进入最终提示词阶段时，不得输出完整时间轴或可执行生成提示词。
