# 动作库

动作库用于把角色行为拆成可见动作，不得在最终提示词中只写动作库名称。

## 基础规则

每个动作必须能写成：起始状态 → 可见变化 → 结束状态。

最终提示词不能写“采用副歌爆发开场动作”，必须写成角色低头、抬眼、迈步、手臂打开、定格等可见动作。

## 分类

### 普通移动

站定、走近、后退、转身、回头、抬头、低头、坐下、起身、伸手、收手、看向镜头、看向远处、绕过物体、从前景后走出、停顿定格。

### 人物情绪动作

低头蓄力、抬眼确认、轻微皱眉、深呼吸、手指收紧、肩膀放松、慢慢转身、靠近光源、停在逆光中、伸手触碰空气、眼神从躲避转为直视。

### 歌曲 / MV 简易舞蹈动作

跟随节奏点头、身体左右律动、单手向外打开、双手从胸前展开、手臂向上划弧、肩部卡点、脚步左右切换、向前迈一步定格、转身半圈回看镜头、副歌强拍抬手、手指指向镜头、麦克风手势。

### 舞台表演动作组合

低头蓄力 → 抬眼 → 向前一步 → 手臂打开 → 半身定格。

### 产品互动动作

手指触碰产品边缘、拿起产品转向镜头、打开盒盖、让产品滑入光线、旋转瓶身、停在 logo 面、用手托起产品。

### 复杂舞蹈动作

复杂舞蹈只建立通道。没有参考音乐或动作参考时，不凭空设计完整编舞。

## 动态级别绑定

- 轻动态：眼神、手势、轻微律动。
- 中动态：走位、手臂打开、轻卡点、转身。
- 高动态：快速走位、强卡点、团舞、跳跃、强拍爆发。
