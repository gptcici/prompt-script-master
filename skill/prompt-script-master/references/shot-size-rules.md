# 景别切换规则

景别用于决定是否需要拆分时间段。每个时间段应有明确景别或景别变化。

## 基本功能

- 远景 / 大远景：交代空间、规模、环境关系。
- 全景：展示人物完整身体、走位、舞蹈或产品与环境关系。
- 中景：兼顾表情、上半身动作和环境。
- 半身近景 / 胸像近景：突出人物状态、演唱、手部动作和情绪。
- 近景 / 特写：突出眼神、道具、产品细节、logo 或情绪爆点。

## 硬规则

- 不要连续两个时间段使用完全相同主景别。
- 相同景别下的连续动作应合并在同一时间段。
- 拆段必须体现景别、焦点、构图、运镜或情绪阶段变化。
- 如果必须保持同一景别，应写“保持同一景别连续完成动作”，不要拆成多个段落。

## 常见递进

- 空间建立：远景 → 全景 → 中景。
- 情绪推进：中景 → 半身近景 → 近景 / 特写。
- 产品广告：全景 / 中景展示环境 → 近景材质 → 特写 logo / 卖点。
- 舞台高光：全景建立舞台 → 中景建立主体 → 半身近景 / 胸像近景完成爆发。

## 检查问题

1. 每段是否有明确景别？
2. 相邻段是否重复同一景别？
3. 景别变化是否服务动作或情绪？
4. 景别是否和运镜、焦点、光线一起写入时间轴？
