# 视频提示词模板

## 统一输出结构

单镜头和多镜头使用同一结构，唯一区别是时间轴分段数量：单镜头 = 1 段，多镜头 = n 段。

```text
【全程总定调】
生成 [时长] 秒，[画幅]，Seedance 2.0 全能参考。
镜头总基调：[风格 + 情绪 + 节奏]。
参考图角色精准锁定：[素材名称] 作为 [参考类型]，属于 [优先级]，锁定 [内容]。
人物核心外形特征：[发型 / 服装 / 体型 / 肤色 / 标志性特征]。
整体场景总设定：[场景描述 + 空间层次 + 氛围基调]。

【时间轴分段】
[段数由分段确认流程决定]

首段：
本段景别：[远景/全景/中景/近景/特写，可写景别变化]
本段运镜状态：[固定/推/拉/摇/跟/环绕/升降，含方向和速度]
焦点变化：[本段焦点对象和切换逻辑]
核心动作：[主体正在做什么]
表情肢体联动：[面部表情 + 头颈肩手/身体重心的联动变化]
人物面光/骨骼光影：[主光方向、骨骼锚点明暗、眼神光]
环境光影变化：[场景光源变化、阴影、色温、亮度]
环境细节补充：[背景、道具、空间层次的具体变化]
本段景深变化：[前后景虚实切换、景深推进或释放]

[中段：9 项同上，不重复角色外貌]

末段：
本段景别：[...]
本段运镜状态：[...]
焦点变化：[...]
核心动作：[...]
表情肢体联动：[...]
人物面光/骨骼光影：[...]
环境光影变化：[...]
环境细节补充：[...]
本段景深变化：[...]

【全局收尾】
整体风格质感：[画幅 / 色彩 / 质感 / 稳定性 / 总体摄影风格]。
全局一致性约束：[主体一致性 / 场景一致性 / 人物连续性 / 禁止项]。
```
