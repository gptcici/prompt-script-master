# 视频提示词模板

## 统一输出结构

单镜头和多镜头使用同一三段式结构，唯一区别是时间轴分段数量：单镜头 = 1 段，多镜头 = n 段。

最终提示词只输出模型可执行内容，不输出解释、评分、自检过程或内部规则说明。

## 强制顺序

`【全程总定调】` 内必须按以下顺序写：

1. 镜头总基调 + 镜头框架
2. 参考图角色精准锁定
3. 人物核心外形特征
4. 整体场景总设定

镜头框架之后，必须立刻写 `参考图角色精准锁定` 和 `人物核心外形特征`。不得在角色锁定和人物核心外形之间插入环境、氛围、道具、色调或情绪描述。人物权重永远高于环境。

## 时间轴写法

`【时间轴分段】` 中，每一段都必须把以下 9 项信息连贯写成一整段自然语言：

```text
本段景别 + 本段运镜状态 + 焦点变化 + 核心动作 + 表情肢体联动 + 人物面光/骨骼光影 + 环境光影变化 + 环境细节补充 + 本段景深变化
```

有几段写几次。九项是内容检查项，不是输出字段。禁止拆成冒号字段列表。

首段需要建立人物主体和关键外形特征；后续段落只写本段变化，保持人物连续性，不重复首段已经建立的角色外貌。

只要提示词描述到角色面部、眼神、表情、眨眼、侧脸、三分之二脸、皮肤、鼻梁、颧骨、下颌线或其他可见面部区域，必须在同一段里写清人物面光/骨骼光影，包括主光方向、鼻梁/眼窝/颧骨/下颌线明暗、眼神光和环境反光。

## 标准模板

```text
【全程总定调】
镜头总基调：[风格 + 情绪 + 节奏]；镜头框架：[景别设计 + 运镜框架 + 镜头叙事功能]。参考图角色精准锁定：[素材名称] 作为 [参考类型]，属于 [优先级]，锁定 [脸部比例 / 五官气质 / 发型 / 身份气质 / 服装轮廓 / 关键材质]，不得变成通用 AI 人物或其他身份。人物核心外形特征：[发型 / 服装 / 体型 / 肤色 / 标志性配饰 / 材质行为]。整体场景总设定：[场景描述 + 空间层次 + 氛围基调 + 真实物理约束]。

【时间轴分段】
首段：[本段景别 + 本段运镜状态 + 焦点变化 + 核心动作 + 表情肢体联动 + 人物面光/骨骼光影 + 环境光影变化 + 环境细节补充 + 本段景深变化，连贯写成一整段自然语言；首段同时建立人物主体和关键外形特征。]

中段：[九项信息同上，连贯写成一整段自然语言；只写本段变化，保持角色、服装、空间和运镜衔接，不重复首段已经建立的角色外貌。]

末段：[九项信息同上，连贯写成一整段自然语言；完成情绪、动作或空间收束。]

【全局收尾】
整体风格质感：[画幅 / 色彩 / 质感 / 稳定性 / 总体摄影风格 / 真实摄影约束]。全局一致性约束：[主体一致性 / 场景一致性 / 人物连续性 / 服装发饰连续性 / 禁止项]。
```

## 禁止输出旧格式

不得在 `【全程总定调】` 第一行写视频基础规格、模型版本、画幅、时长、分段序号或约几秒之类的信息。第一行只能从镜头总基调和镜头框架开始。

不得在 `【时间轴分段】` 中把九项拆成逐行字段。九项必须融合到同一段自然语言镜头描述里。

时长、画幅、模型版本可以作为外部使用备注或用户确认信息处理；除非用户明确要求写入提示词，否则不要混入 `【全程总定调】` 第一句。
