# 最终输出格式稳定规则

## 核心原则

最终输出必须稳定、清楚、可直接复制粘贴。

系统默认不展示内部评分、扣分过程、JSON、自检表或修改过程。

## 统一输出结构

单镜头和多镜头使用同一三段式结构，唯一区别是时间轴分段数量：单镜头 = 1 段，多镜头 = n 段。

```text
【全程总定调】
视频基础规格 + 镜头总基调
参考图角色精准锁定
人物核心外形特征
整体场景总设定

【时间轴分段】
本段景别 + 本段运镜状态 + 焦点变化 + 核心动作 + 表情肢体联动 + 人物面光/骨骼光影 + 环境光影变化 + 环境细节补充 + 本段景深变化

【全局收尾】
整体风格质感
全局一致性约束
```

## 参考素材提醒

最终提示词输出后，必须提醒用户：

> 实际使用时，请加入对应参考素材，例如人物模板图、首帧 / 尾帧、分镜图、场景图、道具图或风格参考图，以提高一致性和生成稳定性。

如果提示词明显依赖参考素材，还要说明优先准备哪些素材。

## 输出简洁规则

最终输出应该是完整提示词，不是解释文章。

除非用户要求，否则不要展示：

- 评分
- 扣分项
- 思考过程
- 多轮修改记录
- 内部草稿

## 最终标准

用户拿到后可以直接复制到 AI 视频模型中使用。
