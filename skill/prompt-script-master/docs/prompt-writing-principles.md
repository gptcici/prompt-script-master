# 提示词写作原则

## 核心原则：动词先行，形容词收束

提示词不能只堆形容词。系统必须先写可见动作、状态变化、空间关系和镜头运动，再用形容词收束情绪或质感。

## 合格写法

不要只写：

```text
她很悲伤。
```

应写：

```text
她低下眼睛，嘴唇轻轻闭合，眉心向内收紧，手指松开又重新握住衣角，显得很悲伤。
```

不要只写：

```text
画面很高级。
```

应写：

```text
主体站在极简空间中央，黑色反光地面映出轮廓，柔和侧光沿服装边缘滑过，背景保持大面积留白，画面显得高级而克制。
```

## 写作顺序

1. 先写主体是谁。
2. 再写主体在哪里。
3. 再写主体正在做什么。
4. 再写镜头如何观看主体。
5. 再写空间、光线、背景如何变化。
6. 最后用少量形容词固定情绪、风格或质感。

## 抽象词处理

用户输入“燃、震撼、高级、孤独、温柔、破碎感、宿命感、电影感”时，系统必须转化为可见画面。

抽象情绪词不能单独作为最终提示词主体。

## 不合格写法

```text
高级电影感，震撼，燃，很好看。
```

问题：主体不明确，场景不明确，没有动作，没有镜头语言，AI 模型不可执行。

## 最终标准

提示词不是文案，而是生成指令。

越具体、越可见、越连续，模型越容易执行。
