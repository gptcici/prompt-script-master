# 默认视频配置

## 默认模型
Seedance 2.0 全能参考。

## 默认画幅
如果用户没有指定，系统可以根据用途默认：

- 横屏电影感：21:9 或 16:9
- 竖屏社媒：9:16
- 方形内容：1:1

如果画幅会影响核心构图，应询问用户。

## 默认时长
单个分镜或单镜头通常 6-15 秒。

低于 6 秒：适合简单转场，不适合复杂动作。
超过 15 秒：需要拆分或压缩。

## 默认摄影机
可使用：

- ARRI Alexa 35
- ARRI Alexa Mini LF
- RED V-RAPTOR
- Sony VENICE
- 电影级数字摄影机

## 默认焦段
- 18-24mm：广角、空间、透视、追逐
- 35mm：人物 + 环境
- 50mm：自然视角
- 85mm：人物英雄近景、情绪推进
- 100mm macro：产品和局部细节
- 135mm：长焦压缩和背景虚化

## 最终标准
默认值服务于画面目的，不机械套用。
