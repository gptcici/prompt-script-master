# Composition Library Assets

Store composition-reference images here after the user uploads them. These images are used only to study shot design, blocking, frame balance, depth arrangement, camera placement, and edit-continuity composition for AIMV / AI short-drama prompts.

Rules:
- Do not treat these images as character identity references.
- Do not copy watermarks, text, logos, creator marks, or exact copyrighted layouts.
- Extract only composition logic: camera height, subject placement, foreground occlusion, leading lines, negative space, frame-within-frame, shot scale, and spatial layering.
- Composition references must always yield to the realism rule and to any uploaded face reference.

Recommended filename pattern:
`comp-01-[short-slug].webp`
