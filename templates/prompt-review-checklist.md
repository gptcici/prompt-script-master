# 提示词自检清单模板

## 使用方式

该清单用于内部自检。默认不展示给用户，除非用户要求查看评分或检查过程。

## 基础自检

```text
【基础自检】
1. 主体是否明确？
2. 场景是否明确？
3. 动作是否可见？
4. 镜头语言是否明确？
5. 时长是否在合理范围？
6. 情绪是否通过动作、镜头、灯光或空间表达？
7. 参考素材优先级是否明确？
8. 一致性要求是否明确？
9. 禁止项是否精准？
10. 是否可直接复制到 AI 视频模型？
```

## 一票否决检查

```text
【一票否决】
- 主体不明确
- 严重逻辑冲突
- 单镜头超过 15 秒
- 单镜头低于 6 秒但承担复杂动作
- 没有动作细节
- 没有镜头语言
- 没有整体风格与基调
- 参考素材冲突
- 首尾帧无法自然衔接
```

## 自动优化检查

```text
【自动优化】
- 抽象形容词是否转化为可见画面？
- 动作是否足够具体？
- 运镜是否有起点、方向和终点？
- 焦点和景深是否清楚？
- 时间轴是否自然？
- 负面提示词是否过多或重复？
- 是否存在过度文学化表达？
```

## 输出判断

```text
90+：优秀，可以输出
85-89：合格，轻微优化后输出
低于85：继续优化
低于70：重写
```

## 最终标准

自检是内部质量控制，不是默认展示内容。
