# 负面提示词模板

## 核心原则

负面提示词只禁止关键异常，不堆砌风格词，不写无意义否定。

## 通用模板

```text
【禁止项】
不要主体漂移，不要身份变化，不要动作断裂，不要场景结构变化，不要焦点错误，不要多出人物或物体，不要字幕、水印、logo、乱码文字。
```

## 人物镜头

```text
不要换脸，不要五官漂移，不要发型变化，不要服装变化，不要肢体变形，不要手指错误，不要表情崩坏，不要焦点偏离人物眼睛，不要字幕、水印、乱码文字。
```

## 演唱会 / 舞台

```text
不要换脸，不要麦克风拿反或变成多支，不要歌手离开舞台指定位置，不要观众进入主舞台，不要舞台结构消失，不要灯光遮住五官，不要服装漂移，不要字幕、水印、乱码文字。
```

## 产品广告

```text
不要产品变形，不要品牌标识错位，不要材质错误，不要多出产品，不要比例变化，不要反光脏乱，不要背景抢主体，不要字幕、水印、乱码文字。
```

## 车辆 / 追逐

```text
不要车辆变形，不要车轮错误，不要行驶方向混乱，不要道路结构变化，不要多出无关车辆，不要速度感断裂，不要镜头抖动过度，不要字幕、水印、乱码文字。
```

## 首尾帧图生视频

```text
不要首帧不匹配，不要尾帧不匹配，不要过程跳变，不要人物身份漂移，不要服装变化，不要场景结构变化，不要光线方向突变，不要多出人物，不要字幕、水印、乱码文字。
```

## 数量建议

- 简单人物镜头：5-8 条
- 一般视频：6-10 条
- 复杂舞台 / 首尾帧 / 多参考：8-12 条

## 最终标准

禁止项要针对当前项目风险，不要机械复制全部模板。
