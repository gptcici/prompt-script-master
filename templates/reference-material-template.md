```text
【参考素材清单】

1. 【素材名称】：【素材类型】
   - 优先级：【必须锁定参考 / 强参考 / 仅参考范围】
   - 主要用途：【锁定身份 / 场景结构 / 风格氛围 / 光线 / 材质 / 动作路径】
   - 必须保持：【不可改变内容】
   - 可以调整：【可电影化调整内容】

2. 【素材名称】：【素材类型】
   - 优先级：【必须锁定参考 / 强参考 / 仅参考范围】
   - 主要用途：【用途】
   - 必须保持：【内容】
   - 可以调整：【内容】
```

## 优先级判断

### 必须锁定参考

用于身份、产品、首尾帧、品牌主视觉等不能变化的内容。

### 强参考

用于场景结构、镜头路径、灯光氛围、视频运动方式等强约束内容。

### 仅参考范围

用于局部材质、颜色、表情、氛围、道具细节等弱约束内容。
