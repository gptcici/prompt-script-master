# 视频提示词模板

## 统一输出结构

单镜头和多镜头使用同一结构，唯一区别是时间轴分段数量：单镜头 = 1 段，多镜头 = n 段。

最终提示词分为「正文提示词」和「负面提示词」两部分。

> **旧版三段式（全程总定调/时间轴分段/全局收尾）已废弃，旧版 9 项时间轴格式已废弃。** 禁止使用旧版结构。

## 正文提示词模板

```text
【正文提示词】
16:9 横屏，[时长]秒，[全程运镜总基调描述]。
@Image1 仅作为人物强参考，锁定[五官/发型/发饰/服装款式/气质]，不作为首帧，不参考原图构图背景，全程为同一位[角色身份]，不生成通用AI模特脸。
[人物核心外形特征：纤细身形、黑色长发高髻、金色花冠配垂落珠链、浅金白色轻纱古装、透明披帛、轻薄衣袖、真实布料层次、清冷安静的东方气质]。
[整体场景总设定：一句话概括大环境与核心元素，如“山顶悬崖边古雅东方凉亭，外侧连接山顶平台与石阶，亭内木质立柱围栏，柱间悬挂轻薄纱幔；悬崖外是辽阔云海与红金夕阳，山风持续稳定吹动”]。

[时间段1]：[景别 + 运镜状态]，[人物核心动作]，[衣发纱幔动态]，[人物光影]，[环境细节]，[景深变化]。首段同时建立人物主体和关键外形特征。

[时间段2]：[景别转景别，运镜状态]，[人物核心动作]，[衣发纱幔动态]，[人物光影]，[环境细节]，[景深变化]。只写本段变化，保持角色、服装、空间和运镜衔接，不重复首段已建立的角色外貌。

[时间段N]：同上7项，完成情绪、动作或空间收束。

([主风格质感]:1.2)，[整体风格质感：真实电影摄影感/写实/无CG插画感/真实布料重量/真实结构/真实光影/自然景深过渡]。
[全局一致性约束：全程人物造型稳定、风向统一、光影逻辑自洽、空间连贯、运镜平稳无跳变]。
```

### 正文提示词规则

1. **语序权重绝对固定**：视频规格 + 参考图 + 人物特征（第一梯队）→ 场景（第二梯队）→ 时间轴（第三梯队）→ 风格 + 一致性（第四梯队）
2. **参考图指令**：必须放在人物特征之前，使用 `@Image1` 标准化格式
3. **人物优先**：参考图指令后立刻写人物特征，不得被环境、氛围、道具隔开
4. **无抽象语义**：不得出现导演创作思路、心理感受、逻辑说明类描述
5. **同类聚合**：衣发动态集中跟在动作后，光影集中放在环境前，景深集中放在段尾
6. **正向描述**：正文只写正向描述，不得出现"禁止、不要、避免、不得"类词汇
7. **权重标记**：全文加权词不超过 5 个，只对核心视觉名词加权

### 时间轴分段格式（每段 7 项，连成一整段自然语言）

```
景别 + 运镜状态 + 人物核心动作 + 衣发纱幔动态 + 人物光影 + 环境细节 + 景深变化
```

## 负面提示词模板

```text
【负面提示词】
人物类：脸部变形、五官漂移、发型错乱、发饰丢失、服装变色、面部平光、塑料磨皮感、光影扁平、五官糊成一团、面部过曝、死黑无细节
动作类：肢体错乱、手部畸形、手指粘连、动作僵硬
场景类：空间跳跃错位、纱幔无重力漂浮、道具错位
风格类：过度磨皮、过度CG感、云海平面贴图、卡通插画感、画面模糊低质
技术类：运镜抖动、景别突变、画幅错误
```

### 负面提示词规则

1. 所有禁止项必须从正文中完全剥离，不得出现在正文中
2. 按类别（人物/动作/场景/风格/技术）分组罗列
3. 每条简短明确，不超过 6 个字为宜
4. 只写模型可能出现的具体问题，不写抽象概念

## 禁止输出格式

以下格式一律不得出现：

- ❌ 旧版三段式：【全程总定调】→【时间轴分段】→【全局收尾】
- ❌ 旧版 9 项时间轴：本段景别 + 本段运镜状态 + 焦点变化 + 核心动作 + 表情肢体联动 + 人物面光/骨骼光影 + 环境光影变化 + 环境细节补充 + 本段景深变化
- ❌ 正文中出现禁止项或否定词（"禁止变脸、服装变色"等）
- ❌ 九项逐行拆开的字段列表
- ❌ 导演创作思路、心理感受、逻辑说明类抽象描述
- ❌ 模糊的参考图描述（"上传定妆图作为参考"）
- ❌ 对长句子、抽象感受、规则类描述加权重标记
