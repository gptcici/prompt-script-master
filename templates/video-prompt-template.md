# 视频提示词模板

## 单镜头模板

```text
【生成规格】
生成【时长】秒，【画幅】，【模型 / 模式】，整体风格为【风格】。

【参考素材说明】
【素材名称】作为【参考类型】，属于【必须锁定参考 / 强参考 / 仅参考范围】，主要参考【具体用途】。

【镜头目标】
本镜头主要表达【主体】在【场景】中的【动作 / 状态 / 情绪 / 目的】。

【时间轴】
0-【时间点】秒：【主体动作 + 镜头变化 + 环境变化】。
【时间点】-【时间点】秒：【主体动作 + 镜头变化 + 环境变化】。
【时间点】-【结束】秒：【ending pose / final frame / 情绪收束】。

【镜头语言与摄影参数】
使用【摄影机 / 镜头】，焦段【焦段】，镜头从【起点】向【终点】进行【运镜方式】，焦点锁定【焦点对象】，背景【虚化程度 / 保留结构】。

【灯光、场景与动态】
【光线方向】，【场景动态】，【空间层次】，【气氛变化】。

【一致性要求】
保持【人物 / 产品 / 场景 / 服装 / 道具】一致，不发生漂移。

【禁止项】
不要【异常1】，不要【异常2】，不要【异常3】，不要字幕、水印、乱码文字。
```

## 多镜头模板

```text
【项目总设定】
生成【总时长】秒，【画幅】，【模型 / 模式】，整体风格为【风格】。核心主体是【主体】，场景是【场景】，整体情绪是【情绪】。

【参考素材优先级】
1. 【素材 A】：【参考类型】，属于【优先级】，锁定【内容】。
2. 【素材 B】：【参考类型】，属于【优先级】，参考【内容】。

【分镜 1】
时长：【秒数】秒。
镜头目标：【目标】。
时间轴：【动作 + 运镜 + 灯光 + 场景变化】。
摄影参数：【摄影机 / 焦段 / 焦点 / 景深】。
禁止项：【本镜头重点禁止项】。

【分镜 2】
时长：【秒数】秒。
镜头目标：【目标】。
时间轴：【动作 + 运镜 + 灯光 + 场景变化】。
摄影参数：【摄影机 / 焦段 / 焦点 / 景深】。
禁止项：【本镜头重点禁止项】。

【镜头之间的连续性要求】
保持【人物身份 / 服装 / 场景方位 / 运动方向 / 光线 / 道具】连续。

【统一一致性要求】
全片保持【主体、场景、风格】一致。

【统一禁止项】
不要【异常1】，不要【异常2】，不要【异常3】，不要字幕、水印、乱码文字。
```
