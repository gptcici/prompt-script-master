# 提示词脚本大师 (Prompt Script Master) V0.9.82

导演助理式 AI 视频提示词生成 Skill，为 Seedance 2.0/2.1、OpenAI image-family、AIMV / MV、图生视频、首尾帧过渡、多参考图融合和工业化分镜提供结构化中文工作流。

当前 README 只描述 **V0.9.82 当前能力**。历史版本说明请查看 `CHANGELOG.md`。

## 当前版本定位

**V0.9.82 是“参考图上传后统一自检 + 人脸完整性 + 输出模式优先”的增强版。**

核心目标：

- 先判断用户输入属于 T2V / I2V / R2V / FLF2V / V2V / Edit / Extend / IMAGE 哪一种模式。
- 只要上传参考图，先做参考图自检，再决定每张图如何使用。
- 参考图清晰可信的内容可以少写；不清晰的内容必须补写；异常内容不得继承。
- 如果参考图多处不清晰，优先建议用户重新生成或上传更清晰的参考图。
- 最终视频提示词默认输出【正文提示词】+【负面提示词】，图片提示词默认使用自然语言段落。

## V0.9.82 当前核心能力

### 1. 参考图上传后统一自检

只要用户上传图片、首帧、尾帧、人物图、服装图、发饰图、场景图、构图图、产品图或参考图，先执行：

```text
reference-intake-self-check.md
  ↓
reference-visibility-gate.md
  ↓
face-integrity-check.md
  ↓
可用性决策：可直接用 / 部分可用需补写 / 异常需重建 / 建议重新生成参考图
  ↓
reference-transfer-contract.md
  ↓
output-length-mode-standard.md
```

参考图使用原则：

- 清晰可信的内容可以作为锁定锚点。
- 不清晰区域必须在提示词中补写。
- 异常色块、黑斑、死黑阴影、畸形结构、皮肤伪影不得继承。
- 人脸、服装、发型/发饰、背景/空间、构图/景别五类中三类及以上不可辨，或主职责区域不可辨时，建议用户重新生成参考图。

### 2. 人脸完整性检查

涉及人物脸、人物近景、特写、锁脸、修脸、三视图、多视角、角色连续或人物参考图时，检查：

- 人脸畸形、五官错位、眼睛大小异常、嘴唇粘连。
- 不协调色块、黑斑、异常阴影、大面积死黑。
- 皮肤塑料感、蜡像感、橡胶感、过度磨皮、肤色断层。
- 不符合生理逻辑的皮肤质感、面部结构和光影关系。

异常内容进入修复或负面约束，不能作为身份锚点继承。

### 3. 模式优先与输出长度控制

先判断任务模式，再选择输出长度：

| 模式 | 使用场景 | 输出原则 |
|---|---|---|
| T2V | 纯文字生成视频 | 完整描述主体、动作、场景、镜头、光影、动力学 |
| I2V | 图生视频 / 让图动起来 | 只写图片看不出来的动作、运镜、光线变化、动力学和保持约束 |
| R2V | 多参考图 / 图文混合 | 先分配每张参考图职责，再生成提示词 |
| FLF2V | 首尾帧过渡 | 首帧控制起点，尾帧控制终点，中间写连续运动路径 |
| V2V / Edit / Extend | 视频编辑 / 续写 | 只写需要改变或延续的变量 |
| IMAGE | 图片、海报、角色图、三视图、关键帧 | 使用自然语言图片提示词，不默认输出视频时间轴 |

长度档位：

- 压缩版：约 80-180 中文字。
- 快速视频版：约 120-260 中文字。
- 标准视频版：约 500-1000 中文字。
- 高控制视频版：约 1000-1600 中文字。
- 密集分镜 / shot list / storyboard：仅用户明确要求时使用，可超过 1600 字。

### 4. 人物镜头强制覆盖规则

只要最终画面出现可识别人物主体，先判断是否需要：

- 人物动作。
- 表情与肢体联动。
- 口型与说唱表演。
- 人物面部光影。
- 衣物、饰品、发丝和环境动态。
- 人物一致性与参考图锁定。

该规则是人物镜头的顶层判断入口，避免人物提示词只有颜值、服装和氛围，没有可执行动作。

### 5. 物理动力学与镜头语言

涉及发丝、布料、披帛、纱幔、雨雪、烟尘、火焰、粒子、水面、道具、接触、碰撞、重力或惯性时，使用物理动力学规则：

```text
动力来源 → 作用对象 → 运动路径 → 材质响应 → 可见结果 → 结束状态 / 消散状态
```

涉及镜头语言、分镜、AIMV、MV、广告片、storyboard 或复杂运镜时，使用镜头任务、景别选择、运镜选择、多镜头语法和轴线连续规则。

### 6. 质量自检与失败修复

支持：

- 提示词格式检查。
- 参考图职责冲突检查。
- 事件密度过载检查。
- 提示词预算分配。
- 人脸、口型、动作、光影、风向、动力学和连续性失败诊断。
- 重拍迭代时区分保留、局部修、换种子、改写提示词和后期修。

## 最终输出标准

视频提示词默认：

```text
【正文提示词】
总设定 + 参考图职责 + 全局光源锚点 + 全局物理动力学锚点 + 镜头时间轴 + 全局一致性

【负面提示词】
人物类 + 动作类 + 动力学类 + 场景类 + 风格类
```

图片提示词默认：

```text
自然语言段落，不默认拆正负面，不输出视频时间轴。
```
