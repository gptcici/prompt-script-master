# Model Behavior Diagnosis 模型行为诊断

## 目的

本文件用于把外部 Seedance 工作流中的“模型机制”思想抽象到 prompt-script-master 中。它不是平台内部架构说明，而是面向提示词写作、审查和失败复盘的经验性工作模型。

它回答的问题：

- 为什么提示词越长越乱？
- 为什么“不要某物”反而会出现某物？
- 为什么参考图压过文字？
- 为什么远景人物、手、文字和小 logo 更容易崩？
- 为什么连续生成几代后身份会漂移？
- 为什么对口型、多人、镜头运动同时出现时更容易失败？

## 启动条件

出现以下任一情况时读取本文件：

```text
提示词审查 / 失败复盘 / 为什么不听话 / 为什么越写越乱 / 为什么参考图压过文字 / 为什么远景脸崩 / 为什么不要某物却出现 / 为什么对口型失败 / 为什么连续生成漂移 / 模型机制 / 模型不稳定
```

## 八个工作机制

### 1. 注意力是预算

提示词里的每个可见元素都在竞争模型注意力。能被画面看到、被镜头拍到、被光照到、被动作证明的词更有价值；空泛评价词会消耗预算但不给模型明确像素目标。

执行原则：

- 关键主体、参考职责、动作、光源、动力学和镜头目的放在前面。
- 删除重复形容词和空泛评价词。
- 每个镜头优先保留最重要的 1 个主目标和 1 个次目标。
- 长提示词不是更稳定，清晰预算才更稳定。

### 2. 模型会向熟悉画面坍缩

模型更容易生成常见组合，如夕阳逆光、雨夜霓虹、暖色灯笼、舞台聚光等。罕见组合需要拆成可理解的真实片场元素，而不是直接堆抽象标签。

执行原则：

- 使用稳定的视觉锚点：光源、机位、材质、场景、动作。
- 稀有设定要拆成熟悉的真实组件。
- 风格必须由颜色、光线、镜头、材质和构图支撑。
- 多镜头需要重复关键风格锚点，防止风格跳变。

### 3. “不要”很弱，正向替代更强

模型会被被提到的概念激活。正文里反复写“不要模糊、不要多手、不要塑料感”可能反而召唤这些概念。

执行原则：

- 正文只写画面中应该出现什么。
- 故障项写入【负面提示词】。
- 把“不要塑料皮肤”改成“皮肤保留自然毛孔、细微肤色变化和环境光反射”。
- 把“不要手崩”改成“双手自然静置，手指数量清晰，指尖轻触道具边缘”。

### 4. 时间是轨迹，不是指令清单

视频模型更容易跟随有因果和惯性的轨迹。一个清晰动力来源加可见结果，通常强于多个互不相连的小指令。

执行原则：

- 动作写成：起点 → 动力来源 → 路径 → 物理结果 → 终点。
- 不要一个镜头塞多个无关动作。
- 道具、烟雾、水面、衣料、头发等动态必须有来源、路径和结束状态。
- 当前 clip 只做当前该完成的事件。

### 5. 错误会累积

连续生成中，小的脸部漂移、服装漂移、道具状态偏差会一代代放大。不要用已经漂移的输出作为唯一身份锚点。

执行原则：

- 连续项目定期回到原始人物参考图。
- 每段提示词保留必要 identity / costume / prop locks。
- clip 不宜过长；脆弱动作拆分。
- observed_end_state 与 planned_end_state 不一致时，下一段从实际结果修正，不要假装计划成功。

### 6. 参考图在重叠区域强于文字

参考图是高密度条件。文字重复描述同一张参考图，可能制造第二套相似但不完全一致的身份、服装或场景指令，导致漂移。

执行原则：

- 参考图负责能看见的静态信息。
- 文字负责参考图无法表达的时间、动作、镜头、声音、约束和非转移项。
- 多图必须先做 reference-transfer-contract。
- 明确写“图 A 只控制脸，不继承服装；图 B 只控制服装，不继承脸”。

### 7. 细节容量随画面面积缩小

远景中的脸、手、logo、文字和小道具占画面很小，无法承载复杂细节。动作越大、细节越小，越容易失败。

执行原则：

- 重要脸部细节给近景或中近景。
- 重要手部、琴弦、产品 logo、首饰细节给单独特写。
- 群像远景不要要求每个人脸都清晰。
- 小文字移到后期，不要让视频模型生成可读长文本。

### 8. 音频和画面互相约束

唱歌、说话、对白、动作声、环境声会影响时间和口型。镜头运动、多人物、长台词和复杂动作会增加对口型难度。

执行原则：

- 对口型时保持脸部较稳定，减少头部大幅转动和复杂运镜。
- 台词短句化。
- 音乐节奏可以作为剪辑和动作时钟。
- 多人说话时明确谁说、谁听、谁只做微反应。

## 诊断表

| 现象 | 可能机制 | 修复杠杆 |
|---|---|---|
| 长提示词反而泛化 | 注意力稀释 | 删除空泛词，重排优先级 |
| 风格跳变 | 熟悉画面簇跳转 | 重复同一风格锚点 |
| 不要的东西出现 | 否定召唤 | 正向替代，负面归位 |
| 动作跳过或糊成一团 | 缺轨迹 | 一因一果一终点 |
| 连续几段后脸变了 | 错误累积 | 回到原始参考图，缩短 clip |
| 参考图和文字打架 | 条件冲突 | 删掉重述，明确非转移项 |
| 远景脸 / 小字 / 小 logo 崩 | 细节面积不足 | 放大，单独给镜头 |
| 对口型失败 | 音画联合约束过载 | 锁脸、短句、减少运镜和多人物 |
