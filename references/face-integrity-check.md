# Face Integrity Check 人脸完整性与生理逻辑检查（V0.9.82）

## 目的

本规则用于人物参考图、图片生成、图生视频、首尾帧、换脸/锁脸、角色三视图、近景、特写和重拍修复。它专门处理人脸异常风险：畸形、不符合生理逻辑、五官错位、不协调色块、黑斑、异常阴影、大面积黑色、皮肤质感不符合生理逻辑等问题。

本规则不是医学诊断，而是视觉生成质量控制。最终提示词中不得继承参考图里的异常脏污、色块、死黑阴影或面部结构错误。

## 启动条件

```text
用户上传人物参考图、角色图、脸部图、首帧或尾帧
用户要求锁脸、保持人物、换装、三视图、多视角、重绘、修脸、修肤色、修阴影
画面为中近景、近景、特写、面部特写或情绪近景
生成结果反馈脸怪、脸脏、黑斑、阴影怪、皮肤假、五官畸形、脸部色块、面部发黑
```

## 人脸自检维度

### 1. 生理结构

检查：

- 双眼位置是否合理，瞳孔方向是否一致。
- 鼻梁、鼻翼、嘴唇、下颌线是否符合正常面部空间关系。
- 面部左右比例是否合理，是否出现脸部扭曲、五官粘连、眼嘴错位、鼻唇塌陷。
- 面部轮廓是否与头部角度、脖颈、肩颈连接一致。

正向锚点：

```text
五官位置符合自然面部结构，眼鼻唇比例协调，脸型边界清楚，头颈连接自然。
```

### 2. 皮肤色调连续性

检查：

- 皮肤是否出现不协调色块、黑斑、脏污式斑块、局部灰绿/紫黑污染。
- 是否出现大面积黑色覆盖脸部但不符合光源方向。
- 肤色是否从亮面到暗面自然过渡，而不是硬边色块。
- 皮肤是否变成塑料、蜡像、橡胶、油腻或过度磨皮。

正向锚点：

```text
皮肤色调连续自然，明暗过渡服从单一光源，保留真实细节和柔和肤质，不出现脏污色块或异常黑斑。
```

### 3. 面部阴影逻辑

检查：

- 阴影是否来自明确主光源。
- 眼窝、鼻翼、颧骨、下颌阴影是否符合面部骨骼结构。
- 是否出现无来源黑影、脸颊大块死黑、眼周异常黑洞、鼻梁断裂阴影。
- 阴影边缘是否自然，是否把面部结构切碎。

正向锚点：

```text
面部阴影由明确主光源产生，鼻梁、眼窝、颧骨和下颌线明暗关系自然，暗部保留皮肤细节。
```

### 4. 眼睛与嘴部完整性

检查：

- 眼睛是否大小合理，眼白、瞳孔、眼神光是否自然。
- 嘴唇开合是否符合情绪或口型，不出现粘连、裂开、错位。
- 唱歌/说话/对口型时，下颌和唇部动作是否稳定。

正向锚点：

```text
双眼清晰有自然眼神光，瞳孔方向一致；嘴唇边界清楚，开合幅度符合表情和口型。
```

### 5. 参考图异常继承控制

如果参考图本身存在异常：

- 不继承脸上黑斑、脏污色块、异常阴影、大面积死黑。
- 不继承畸形五官、脸部扭曲、错误光斑、AI伪影。
- 只抽象继承可用身份信息：脸型比例、五官气质、年龄感、妆容方向、发型轮廓。
- 需要在正文补写“自然重建”锚点，在负面提示词写异常故障项。

## 与正文 / 负面提示词的分工

正文写正向目标：

```text
面部结构自然，五官比例协调，皮肤色调连续，明暗过渡服从主光源，暗部保留皮肤细节，眼神光自然。
```

负面提示词写故障：

```text
人物类：脸部畸形、五官错位、眼睛大小异常、嘴唇粘连、脸上不协调色块、黑斑、异常阴影、大面积死黑、皮肤塑料感、蜡像感、橡胶感、过度磨皮、肤色断层、脏污斑块。
```

## 修复 / 重拍判断

当用户反馈“脸脏 / 脸黑 / 色块 / 阴影怪 / 皮肤假 / 脸畸形”时，不要只加强美颜。先判断：

1. 是参考图本身异常被继承。
2. 是光源方向不清导致阴影错误。
3. 是近景面光不足。
4. 是提示词过长导致脸部注意力不足。
5. 是运镜或动作过强导致面部稳定性下降。
6. 是风格词过度导致皮肤变成塑料/蜡像。

然后只改一个主变量：参考图职责、面部正向锚点、光源、运镜稳定性、预算分配或负面提示词。


## 严重异常时的参考图重生建议

如果参考图中的人脸承担 identity / lock-face / 三视图 / 角色连续职责，但出现以下情况之一，不应继续把该图作为强锁脸参考，应建议用户重新生成或上传更清晰的人脸参考图：

- 五官不可辨、脸部过小、被遮挡、低清、过曝或大面积死黑。
- 面部存在明显畸形、五官错位、眼鼻唇关系不符合生理逻辑。
- 脸部有大面积黑斑、异常色块、脏污式斑块、压缩噪点或不受光源解释的死黑阴影。
- 皮肤质感明显不符合生理逻辑，例如塑料、蜡像、橡胶、油泥、过度磨皮或肤色断层。

建议用户重新生成时，应给出可执行要求：正面或三分之二侧脸、五官清楚、面部无遮挡、主光源明确、暗部保留皮肤细节、无异常色块、无死黑阴影、皮肤质感自然。
