# 演唱会 / Live MV 规则

用于舞台、唱歌、演唱会、Live MV、偶像舞台、乐队现场和大型表演类视频提示词。只吸收可泛化导演规则，不绑定任何具体歌曲、艺人、视频时间戳或逐秒分镜。

## 触发条件

当用户输入包含以下任一方向时读取本文件：

- 演唱会、现场、live、舞台、T 台、主舞台、延伸台
- 歌手、主唱、乐队、偶像、唱跳、舞蹈表演
- 副歌、高潮、爆点、大合唱、观众互动、灯海、荧光棒
- 大场面、体育场、万人现场、烟花、彩带、LED 屏、追光

## 不入库内容

不要把参考视频里的以下内容直接写入规则或最终提示词：

- 本地文件路径、原视频文件名、特定艺人或具体曲名
- 逐秒分镜表、固定时间戳、固定镜头编号
- 只属于某个视频的一次性特效或剧情安排
- 无法泛化的歌词节点、歌词文本和表演细节

只能把参考拆解中的结构性规律提炼为项目规则、镜头组合、动作方案和质量门槛。

## 三线叙事结构

演唱会 / Live MV 不能只拍主唱。必须同时维护三条视觉线：

1. 主体线：主唱、歌手、舞者或乐队成员。
   - 用中景、近景和特写锚定人物情绪、唱歌动作和表演强度。
   - 必须写清可见动作，例如奔跑、抬手、转身、张开双臂、看向观众、靠近麦克风。

2. 观众线：现场观众、合唱、挥手、灯海、荧光棒或手环。
   - 观众不是背景，而是情绪放大器。
   - 观众镜头用于桥接主唱镜头、强化共鸣、表现副歌和高潮。

3. 空间线：主舞台、T 台、延伸台、LED 屏、灯光、烟雾、彩带、烟花和场馆规模。
   - 用大全景、航拍感、摇臂高位或拉远镜头建立空间结构和视觉规模。
   - 必须说明人物与舞台、观众、灯光主视觉之间的空间关系。

最终时间轴中应交替使用三线，不要连续多个段落只写同一主体。

## 音乐驱动剪辑规则

演唱会 / Live MV 的镜头变化必须服从音乐结构：

- 鼓点、重拍、强拍：优先作为切镜点或动作落点。
- 主歌：节奏相对稳定，多用中景跟拍、表情近景和乐器 / 动作细节。
- 预副歌：逐渐缩短镜头段落，增加推近、快切、灯光增强和动作蓄力。
- 副歌 / 高潮：必须提升规模感，交替使用人物近景、观众反应、灯海大全景和英雄式舞台镜头。
- 尾奏 / 收尾：节奏可以放慢，用观众合唱、航拍拉远、彩带 / 烟花或一个视觉记忆点收束。

如果用户没有提供音乐风格、节奏速度、使用段落或情绪方向，不得直接设计确定动作和完整时间轴；只能给临时动态判断并询问关键音乐信息。

## 景别呼吸规则

演唱会镜头需要使用“大-小-大”的呼吸节奏：

1. 大：用大全景 / 航拍感建立体育场、舞台和观众规模。
2. 小：切到主唱、乐器、观众脸部或手部特写，制造亲密感。
3. 大：回到灯海、舞台全貌、T 台纵深或彩带烟花，释放情绪。

禁止长时间只使用主唱中景，禁止连续多个时间段都是相同主景别。

## 观众作为视觉系统

观众在演唱会提示词中应承担两种功能：

- 情绪功能：欢呼、合唱、举手、拥抱、落泪、跟随节奏挥动。
- 视觉功能：灯海、荧光棒、发光手环、波浪式颜色变化、观众区星空感。

写入时间轴时，观众要么作为前景 / 背景动态，要么作为切换镜头主体。不要只写“观众很多”或“气氛热烈”。

## 机位协同规则

演唱会镜头推荐使用多机位协同，但最终提示词中仍要写成可执行画面语言：

- 航拍感 / 高位俯拍：用于建立规模、展示 T 台几何、灯海和场馆空间。
- 摇臂 / 高位横移：用于打开舞台全貌、制造英雄姿态、从人群扫向主唱。
- 稳定器 / 斯坦尼康跟拍：用于跟随主唱走位、奔跑、靠近观众或穿越 T 台。
- 手持近景：用于观众表情、鼓手 / 乐器细节、主唱临场感。
- 固定机位：只作为节奏锚点，不作为整段主视觉。

单个时间段最多组合 2-3 个主要运镜动作。不要堆叠过多运镜。

## 演唱会镜头组合公式

以下公式只用于构思与沟通，最终提示词必须拆成具体时间轴语言。

### 开场建立公式

航拍 / 高位大全景 → 观众特写 → 舞台中景 / 主体登场

作用：先建立规模，再拉近情绪，最后锁定表演主体。

### 高潮爆发公式

主唱特写 → 观众反应 → 灯海大全景 → 逆光英雄镜头

作用：个人情绪、群体共鸣、视觉奇观和象征性定格逐层放大。

### 节奏推进公式

乐器 / 动作细节 → 主唱中景 → 观众手部或面部特写 → 舞台全景

作用：用不同主体承接节奏，避免视觉单调。

### 结尾收束公式

主唱中景动作 → 观众合唱特写 → 彩带 / 烟花 / 灯海全景 → 视觉记忆点或黑场

作用：把情绪从人物释放到全场，再用记忆点收束。

## 高潮镜头优先候选

当用户提到副歌、高潮、爆点、大场面或情绪爆发时，优先从以下候选中选 2-4 个推荐给用户：

- 逆光剪影英雄镜头：低角度或摇臂高位，主唱张开双臂，背后强光和烟雾形成轮廓。
- 灯海星空镜头：高位或航拍感俯拍，观众灯光形成波浪或星海。
- T 台孤立人物镜头：主唱站在 T 台远端，被观众环绕，突出个体与巨大群体的对比。
- 观众合唱桥接镜头：观众面部、手部和荧光棒近景，用于连接主唱镜头和大全景。
- 彩带 / 烟花庆典镜头：副歌后段或结尾使用，作为视觉奖励。

## 动作设计规则

音乐信息明确后，动作方案应按动态级别设计：

- 轻动态：站定演唱、抬眼、握麦、轻微点头、单手打开、看向观众。
- 中动态：沿 T 台走位、转身回看、抬手带动观众、半圈环绕、靠近舞台边缘互动。
- 高动态：奔跑、跳跃落地、强拍张开双臂、快速走位、群舞卡点、与乐队成员互动。

所有动作都要拆成可见身体行为。不要只写“激情表演”“燃炸全场”。

## 质量检查

演唱会 / Live MV 最终提示词必须检查：

1. 是否同时有主体线、观众线和空间线。
2. 是否根据音乐段落安排节奏变化。
3. 是否在时间轴内逐段写入光线、运镜、焦段 / 镜头类型、景别和焦点。
4. 是否使用大-小-大的景别呼吸。
5. 是否把观众作为情绪和视觉系统，而不是背景人群。
6. 是否避免过多堆叠航拍、环绕、变焦、旋转等复杂运镜。
7. 是否把经典镜头名称拆成可执行镜头语言。
8. 是否删除了具体参考视频、艺人、曲名、时间戳和沟通话术。

## V0.6 补充：女歌手 / 主唱演唱会细节规则

本节从大型室内演唱会动作与镜头拆解中抽象而来，只保留通用导演方法，不保留具体来源。

### 动作服务情绪

流行女歌手或主唱类演唱会不一定依赖大幅度运动。脚本必须允许“静态中的动态”：人物动作较克制，但通过握麦、视线、手势、站位、服装流动、灯光变化、LED 动态和镜头推进形成画面变化。

### 章节感

演唱会段落可以用造型、光色、舞台视觉和动作密度区分章节。生成脚本时不要把整支视频写成同一种灯光和同一种动作。

### 主唱常用动作路径

- 抒情：稳定站立 / 坐姿，胸前握麦，闭眼，低头，手按胸口，镜头缓慢推近或拉远。
- 推进：缓步走位，侧身看向观众，单手外展，灯光逐渐增强，镜头从中景进入中近景。
- 高潮：仰头演唱，双臂张开或高举，逆光与烟雾形成轮廓，随后切舞台全景和观众反应。
- 互动：举麦朝向观众，微笑回应，走到舞台边缘或延伸台，与观众灯海形成前后景关系。

### 镜头比例倾向

抒情与精致舞台段落优先中景 / 中近景 / 特写，不要过度使用航拍和快速运动。高潮和开结尾再使用大全景、摇臂和观众灯海。
