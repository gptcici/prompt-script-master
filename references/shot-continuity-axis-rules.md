# Shot Continuity Axis Rules 镜头连续与轴线规则

## 目的

保持多镜头和连续片段中的屏幕方向、人物视线、运动方向、光源方向、风/动力学方向、道具位置和空间关系连续。

## 启动条件

- 多镜头时间轴。
- 首帧 / 尾帧过渡。
- 下一段继续、第二段、连续剧情、系列 MV。
- 两人对话、群像、舞台走位、动作场面。
- 用户反馈“方向变了 / 位置跳了 / 镜头怪 / 不连贯”。

## 连续锚点

内部必须确认：

```text
screen_direction：主体在画面中朝哪边移动或看向哪边。
eyeline：人物视线方向和对视关系。
axis：对话、打斗、行走或舞台空间的轴线。
entry_exit：人物从哪边入画、从哪边出画。
light_direction：主光源方向是否延续。
physical_direction：风、雨、烟、云、水、衣物、粒子的方向是否延续。
prop_position：道具在人物哪一侧、是否被拿起/放下/移动。
start_end_handoff：上一镜头终点是否成为下一镜头起点。
```

## 轴线规则

1. 对话镜头保持人物左右关系，不无故反转。
2. 行进镜头保持屏幕方向，不让人物上一镜头向右走、下一镜头无理由向左走。
3. 打斗/舞蹈/队形镜头保持前后左右关系，除非明确写出绕场或换位。
4. 主光源方向不随切镜随机改变。
5. 风、雨、烟、云、衣摆、发丝方向不随切镜随机改变。
6. 焦点转移只在明确主体之间发生，不跨越无关物体。
7. 首帧控制起点，尾帧控制终点，中间过程不得提前抵达尾帧。

## 修复方向

如果生成结果方向混乱，优先修：

```text
屏幕方向
人物视线
入画/出画方向
主光源方向
风/动力学方向
道具状态
上一镜头终点与下一镜头起点
```

不要只加“保持一致”，要写清具体哪一项一致。
