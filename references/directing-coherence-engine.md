
# Directing Coherence Engine 导演一致性引擎

## 目的

本规则用于把“好看、电影感、氛围感、大片感、情绪强”等模糊要求，转化为可执行的导演决策。它不是替代人物镜头强制覆盖规则，而是在每个镜头生成前先确定“这个镜头到底要让观众感受到什么”。

## 启动条件

出现以下任一情况时读取本规则：

```text
剧情短片、AIMV、MV、古风 MV、情绪短片、连续剧情、分镜、故事板、导演感、电影感、氛围感、镜头语言、强情绪、叙事、转折、揭示、告别、决心、压迫、孤独、梦幻、史诗、商业广告、产品大片
```

## 核心原则

一个镜头只能有一个核心意图。镜头、景别、运镜、光、颜色、表演、声音、剪辑点都必须服务同一个意图。

错误方向：

```text
电影感、氛围感强、镜头高级、光影漂亮。
```

正确方向：

```text
这一镜头的意图是让观众感到女主的决心开始动摇，所以镜头缓慢推近，侧后方冷月光压住半张脸，眼神先避开再回到镜头，手指停在琴弦上，风声降低，只留下琴弦余音。
```

## 导演读取五问

在复杂镜头或剧情片段中，内部先回答五个问题。用户不要求时，不必完整展示。

1. **场景功能**：这一镜头是介绍、推进、转折、揭示、告别、压迫、释放，还是收束？
2. **情绪转折**：镜头开始和结束之间，情绪从什么变成什么？
3. **观众位置**：观众是在角色内心、旁观、压迫角色，还是被角色凝视？
4. **权力关系**：谁掌握主动，谁被动，权力是否在镜头中发生转移？
5. **潜台词**：角色没有说出口的真实欲望或矛盾是什么？

## 镜头意图句

复杂镜头必须先形成一句内部意图句：

```text
镜头意图：让观众感到 [情绪 / 信息 / 关系] 从 [起点] 变为 [终点]。
```

示例：

```text
镜头意图：让观众感到她从强撑镇定转为被琴声击中。
镜头意图：让观众先以为环境安全，再发现暗处有人注视。
镜头意图：让观众看到产品材质的高级感从触感而不是旁白中被证明。
```

## 工具统一表

| 工具 | 承担什么 | 写法重点 |
|---|---|---|
| 景别 | 亲密、距离、孤立、规模 | 近景进入内心，远景显示环境压迫或孤独 |
| 机位高度 | 权力和同理心 | 低机位强化主体，高机位削弱主体，平视建立共情 |
| 焦段感 | 心理空间 | 长焦压缩孤立，广角扩空间或制造压力 |
| 运镜 | 观众冲动 | 推近=靠近真相/情绪，拉远=疏离/揭示，静止=屏住呼吸 |
| 光影 | 情绪暴露 | 光源方向、明暗比例、色温、阴影边界表达安全/威胁/距离 |
| 表演 | 潜台词 | 用一个可见动作表现角色没说出口的东西 |
| 声音/节奏 | 呼吸和压力 | 静音、余音、环境声密度、节拍点控制镜头重量 |
| 剪辑点 | 情绪释放或悬停 | 在动作落点、眼神变化、光影变化或声音落点切换 |

## 场景类型快速映射

### 亲密情绪
- 景别：中近景 / 近景 / 特写。
- 镜头：稳定或极慢推近。
- 光：柔和可见动机光，保留眼神光和暗部层次。
- 表演：眼神、呼吸、指尖、肩颈等微动作。

### 对峙 / 权力变化
- 景别：中景到近景，人物分侧站位。
- 镜头：保持轴线，权力转移时改变高度或推进。
- 光：冷暖分离或一侧压暗。
- 表演：一个角色保持静止，另一个角色失去控制的细小动作。

### 揭示 / 发现
- 景别：先遮挡后揭示。
- 镜头：横移、推近、拉焦或光线打开信息。
- 光：门缝、窗光、灯笼、月光、反射等有动机的揭示光。
- 表演：反应是重点，角色认知变化比物体本身更重要。

### 决定 / 转折
- 景别：从环境压迫进入人物近景。
- 镜头：缓慢推近或静止停顿。
- 光：一侧光加强或暗部吞没背景。
- 表演：一个明确动作作为承诺，如握紧、松手、抬眼、转身。

### 产品 / 材质证明
- 景别：中景建立使用场景，近景或微距证明材质。
- 镜头：锁定或慢推，避免过多动作。
- 光：真实可见光源制造材质高光、边缘光和反射。
- 表演：手部动作简单，产品形状、Logo、材质稳定。

## 与其他规则的关系

- 本规则先确定镜头意图。
- 人物镜头仍执行《人物镜头强制覆盖规则》。
- 动作写法读取 `motion-performance-contract.md`。
- 光影写法读取 `lighting-intention-contract.md`。
- 多人调度读取 `character-blocking-contract.md`。
- 连续项目仍读取 `sequence-state-capsule.md`。
