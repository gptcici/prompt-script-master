# 脚本学习库索引

本库从用户提供的演唱会动作拆解、MV结构拆解等资料中提取可泛化导演方法。它不是案例复刻库，而是脚本结构、动作、镜头、剪辑和光影的抽象参考库。

## 已吸收的抽象能力

1. 演唱会主唱动作分级：轻动态、中动态、高动态。
2. 舞台镜头三线结构：主体线、观众线、空间线。
3. 演唱会景别呼吸：大全景建立、近景聚焦、再回大全景释放。
4. MV 五幕结构：引入、展开、转折、高潮、释放。
5. 现实锚点 + 多空间幻想 / 记忆 / 对照结构。
6. 音乐段落对应剪辑密度：主歌慢、预副歌加速、副歌快、尾奏收束。
7. 光影情绪映射：追光孤独、暖光亲密、火光决裂、自然光轻松、逆光梦幻、烟雾光束舞台化。
8. 库内容隔离：所有参考资料只抽象出方法，不输出来源内容。
