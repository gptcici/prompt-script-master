# Animation Camera Grammar 动画 / 手绘 / 2D 镜头语法

## 目的

当用户要求 2D、动漫、手绘分镜、漫画、storyboard、动画感、非写实视频时，不应过度使用真实摄影器材语言。应改用动画制作中的层、帧、画面调度和动作强调语言。

## 启动条件

- 用户要求动漫、二次元、2D、手绘、漫画、分镜图、storyboard、cel animation、anime MV。
- 用户上传或要求铅笔分镜、漫画格、手绘设定图。
- 用户要非写实但有镜头感的画面。

## 避免过度使用

```text
真实焦段、传感器、真实镜头型号、过度 bokeh、纪录片手持、写实浅景深、真实摄影机轨道描述
```

这些会把 2D 画面拉向伪 3D 或伪写实。

## 推荐语言

```text
多平面层次
前景层移动更快
背景层缓慢卷动
角色 cel 层保持清晰
held frame 停帧
impact frame 冲击帧
smear 动作拖影
speed lines 速度线
on twos / on ones 的节奏感
镜头内构图推进
角色轮廓线稳定
阴影块面清晰
```

## 2D 运镜写法

真实摄影：

```text
85mm lens, shallow depth of field, handheld camera push-in
```

动画改写：

```text
中近景手绘动画镜头，角色 cel 层保持清晰，背景云纹层缓慢向后卷动，前景竹叶层以更快速度横向掠过，形成多平面推进感；画面在她抬眼的瞬间短暂停帧半拍，再用细短发丝拖影表现情绪回弹。
```

## 动画景别原则

- 用“画面层次、角色层、背景层、前景层”替代真实镜头器材。
- 角色脸部和轮廓线优先稳定。
- 动作强时可用拖影、速度线、冲击帧，而不是写实运动模糊。
- 分镜图应使用简洁镜头说明，不写过多摄影设备细节。

## 与 storyboard 的关系

Storyboard 输出优先写：景别、构图、角色动作、镜头方向、转场意图和情绪变化。不要把最终视频提示词的复杂光影和动力学全部塞进每个分镜格。
