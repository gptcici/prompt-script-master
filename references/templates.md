# Skill Templates

## 单镜头输出结构

```text
【生成规格】
生成【时长】秒，【画幅】，【模型 / 模式】。

【参考素材说明】
【素材名称】作为【参考类型】，属于【优先级】，主要参考【用途】。

【整体目标与风格基调】
本镜头主要表达【主体】在【场景】中的【动作 / 状态 / 情绪 / 目的】，整体风格为【风格】，情绪基调为【基调】。

【时间轴】
0-【时间点】秒：【衔接关系 + 主体动作 + 场景动态 + 光线变化 + 摄影机 / 焦段 / 镜头类型 + 运镜方式 + 景别变化 + 焦点控制 + 情绪节奏】。
【时间点】-【结束】秒：【衔接关系 + 主体动作 / ending pose + 场景动态 + 光线变化 + 摄影机 / 焦段 / 镜头类型 + 运镜方式 + 景别变化 + 焦点控制 + 情绪收束】。

【全局摄影基调】
整体保持【画幅 / 色彩 / 质感 / 稳定性 / 景深倾向】。注意：这里只写全局一致的摄影基调，具体场景、光线、运镜、焦段、焦点必须已经写入每个时间轴段落。

【一致性要求】
保持【人物 / 产品 / 场景 / 服装 / 道具】一致。

【禁止项】
不要【异常1】，不要【异常2】，不要字幕、水印、乱码文字。
```

## 时间轴段落模板

```text
【起止时间】：【开场建立 / 衔接上一段 / 镜头切到 / 从上一段继续】，【主体正在做什么】，【场景 / 背景 / 道具 / 空间层次如何变化】，【光线 / 反光 / 阴影 / 色温 / 亮度如何变化】，【摄影机 + 焦段或镜头类型 + 运镜方式】，【景别从什么变到什么 + 构图】，【焦点锁定或转移到哪里 + 景深状态】，【情绪推进或收束】。
```

## 禁止的结构

不要把以下内容作为主执行板块单独写出：

```text
【镜头语言与摄影参数】
【灯光、场景与动态】
```

这些内容必须嵌入每一个时间轴段落。全局段落只能用于补充统一基调，不能替代时间轴。

## 参考素材说明

```text
【参考素材清单】
1. 【素材名称】：【素材类型】
   - 优先级：【必须锁定参考 / 强参考 / 仅参考范围】
   - 主要用途：【用途】
   - 必须保持：【不可改变内容】
   - 可以调整：【可调整内容】
```

## 负面提示词原则

禁止项只写关键异常。一般 5-10 条，复杂项目最多约 12 条。


## 参考库隔离输出模板

最终提示词中的【参考素材说明】只能描述用户当前上传素材在本项目中的功能，不得描述内部库来源。

允许：

```text
参考素材仅用于锁定人物动作节奏、舞台光影方向和镜头构图方法，最终画面以当前项目设定为准。
```

禁止：

```text
参考某某MV / 某某演唱会 / 某某歌手 / 某个文件名 / 某个时间戳。
```
