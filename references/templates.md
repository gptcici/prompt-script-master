# Skill Templates

> **注意**：最终提示词输出格式默认采用 `optimizer-v5/output-templates.md` 的 V5 模板。本文件中的单镜头输出结构为归档参考格式。

## 单镜头输出结构（归档）

```text
【生成规格】
生成【时长】秒，【画幅】，【模型 / 模式】。

【参考素材说明】
【素材名称】作为【参考类型】，属于【优先级】，主要参考【用途】。

【整体目标与风格基调】
本镜头主要表达【主体】在【场景】中的【动作 / 状态 / 情绪 / 目的】，整体风格为【风格】，情绪基调为【基调】。

【时间轴】
首段：【本段景别 + 本段运镜状态 + 焦点变化 + 核心动作 + 表情肢体联动 + 人物面光/骨骼光影 + 环境光影变化 + 环境细节补充 + 本段景深变化】。
末段：【本段景别 + 本段运镜状态 + 焦点变化 + 核心动作 + 表情肢体联动 + 人物面光/骨骼光影 + 环境光影变化 + 环境细节补充 + 本段景深变化】。

【全局摄影基调】
整体保持【画幅 / 色彩 / 质感 / 稳定性 / 景深倾向】。注意：这里只写全局一致的摄影基调，具体场景、光线、运镜、焦段、焦点必须已经写入每个时间轴段落。

【一致性要求】
保持【人物 / 产品 / 场景 / 服装 / 道具】一致。

【禁止项】
不要【异常1】，不要【异常2】，不要字幕、水印、乱码文字。
```

## 时间轴段落模板（归档）

```text
【起止时间】：【本段景别】【本段运镜状态】【焦点变化】【核心动作】【表情肢体联动】【人物面光/骨骼光影】【环境光影变化】【环境细节补充】【本段景深变化】。
```

## 禁止的结构

不要把以下内容作为主执行板块单独写出：

```text
【镜头语言与摄影参数】
【灯光、场景与动态】
```

这些内容必须嵌入每一个时间轴段落。全局段落只能用于补充统一基调，不能替代时间轴。

## 参考素材说明

```text
【参考素材清单】
1. 【素材名称】：【素材类型】
   - 优先级：【必须锁定参考 / 强参考 / 仅参考范围】
   - 主要用途：【用途】
   - 必须保持：【不可改变内容】
   - 可以调整：【可调整内容】
```

## 负面提示词原则

禁止项只写关键异常。一般 5-10 条，复杂项目最多约 12 条。
