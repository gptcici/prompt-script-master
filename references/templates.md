# Skill Templates

> **注意**：最终提示词输出格式默认采用 `templates/video-prompt-template.md` 的统一 2 段式结构（正文提示词 + 负面提示词）。本文件只保留结构参考。

> **旧版三段式（全程总定调/时间轴分段/全局收尾）已废弃，旧版 9 项时间轴格式已废弃。**

## 当前输出结构

```text
【正文提示词】
视频规格 + 全程运镜总基调
@Image1 参考图标准化指令
人物核心外形特征
整体场景总设定

【时间轴分段】
每段把 7 项检查内容融合为一整段自然语言：
景别 + 运镜状态 + 人物核心动作 + 衣发纱幔动态 + 人物光影 + 环境细节 + 景深变化

[权重标记] 整体风格质感 + 全局一致性约束

【负面提示词】
人物类：脸部变形、五官漂移、发型错乱、发饰丢失、服装变色
动作类：肢体错乱、手部畸形、手指粘连、动作僵硬
场景类：空间跳跃错位、纱幔无重力漂浮、道具错位
风格类：过度磨皮、过度CG感、云海平面贴图、卡通插画感、画面模糊低质
技术类：运镜抖动、景别突变、画幅错误
```

## 时间轴段落模板

```text
[时间段N]：[景别 + 运镜状态]，[人物核心动作]，[衣发纱幔动态]，[人物光影]，[环境细节]，[景深变化，连贯写成一整段自然语言]。
```

## 禁止的结构

不要把以下内容作为主执行板块单独写出：

```text
【镜头语言与摄影参数】
【灯光、场景与动态】
```

这些内容必须嵌入每一个时间轴段落。全局段落只能用于补充统一基调，不能替代时间轴。

## 禁止的内容

- 正文中出现"禁止""不要""避免""不得"等否定词（应放入负面提示词）
- 导演创作思路、心理感受、逻辑说明等非视觉化抽象描述
- 衣发动态、光影效果分散在多个位置重复描述
- 把 7 项拆成逐行字段列表
- 模糊的参考图描述（如"上传定妆图作为参考"）
- 对长句子、抽象感受、规则类描述加权重标记
