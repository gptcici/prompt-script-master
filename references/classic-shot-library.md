# 经典镜头库

经典镜头库用于高光、核心、高潮、副歌、爆点、情绪爆发、产品 hero shot、剧情反转和首尾帧关键转折。经典镜头名称只用于用户沟通，最终提示词必须拆成可执行的运镜、景别、焦点、光线和动作。

## 迈克尔·贝式低角度环绕英雄镜头

适合标签：主角登场、副歌爆发、舞台高光、英雄感、动作高潮、视觉爆点、广告大片感、强光背景、烟雾逆光、爆炸感灯光。

核心构成：低角度仰拍 + 摄影机实体推近 + 半环绕 / 弧线移动 + 背景运动视差 + 逆光 / 轮廓光 / 爆发式灯光。

## 希区柯克变焦心理失衡镜头

别名：Dolly Zoom、Vertigo Shot、眩晕镜头、推拉变焦。

适合标签：心理失衡、真相揭示、情绪崩塌、恐惧袭来、空间压迫、剧情转折、世界观坍塌。

核心构成：摄影机移动方向 + 焦段反向变化 + 主体尺寸稳定 + 背景透视异常变化 + 焦点锁定主体。

## 其他跨类型经典镜头

- 斯诺里主观失衡镜头：人物相对稳定、背景剧烈变化，用于失控、迷幻、心理崩溃。
- 鞭甩摇镜转场：快速摇镜、运动模糊、落到新主体，用于卡点和快速转场。
- 甩变焦强调镜头：快速变焦聚焦目标，用于发现、惊吓、强拍强调。
- 拉焦揭示镜头：焦点从一个主体转移到另一个主体，用于线索和关系揭示。
- 荷兰角心理倾斜镜头：画面倾斜，用于不安、危险和心理偏移。
- 摇臂升降揭示镜头：通过升降打开空间，用于开场、舞台规模和宏大场面。
- 一镜到底沉浸跟拍：连续镜头推动复杂调度，用于追逐、舞蹈和沉浸叙事。
- 分焦深焦关系镜头：前景和背景同时清晰，用于双主体张力和监视感。

## 导演风格 / 商业镜头组

- 王家卫式慢门 / 拖影情绪镜头：霓虹、雨夜、慢门拖影、情绪游离。
- 库布里克式对称构图：中心构图、单点透视、稳定压迫感。
- 斯皮尔伯格式推近反应镜头：推近人物反应特写，突出敬畏、恐惧或发现。
- 德帕尔马式分屏 / 监视感镜头：分屏、窥视、监控、双线并行。
- 塔可夫斯基式长镜头空间流动：慢节奏、自然元素、时间流动和空间沉浸。
- 诺兰式时间压迫镜头：交叉推进、倒计时感、多线压迫。
- MV 卡点镜头组：甩镜、甩变焦、快速推近、强拍定格。
- 产品广告 hero shot 镜头组：产品显露、材质高光、旋转展示、logo / packshot 定格。

## 使用规则

每次只推荐 2-4 个最适合的镜头方案。用户确认后，把镜头名拆成基础运镜、景别变化、焦点对象、光线变化和情绪作用写入时间轴；最终提示词中不要保留经典镜头名称。
