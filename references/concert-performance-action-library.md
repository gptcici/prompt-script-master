# 演唱会人物动作库

用于演唱会、Live MV、舞台独唱、唱跳、偶像舞台、AIMV 歌手镜头和音乐短片。只提供通用动作设计，不输出库名或动作标签。

## 使用原则

- 动作必须服务音乐情绪，不为炫技而动。
- 女歌手 / 主唱类画面通常以中景、近景和表情特写承载情绪；动作幅度可小，但必须清楚可见。
- 动作设计应从“身体状态、握麦方式、手势、头部视线、互动对象”五层组合。
- 缺少音乐风格、节奏和段落时，只能给候选动作方向，不得锁死具体动作。

## 动作强度分级

### 轻动态

用于慢歌、内心独白、抒情段、空灵段、孤独段。

可选动作：

- 舞台中央稳定站立，身体轻微侧向。
- 坐在高脚凳或低台上，双手握麦贴近胸前。
- 低头吟唱，随后慢慢抬头。
- 闭眼唱完一句，眉眼保持克制。
- 一只手轻按胸口，另一只手稳定握麦。
- 视线从镜头移向舞台远处，形成回忆感。

### 中动态

用于预副歌、情绪推进、舞台走位、观众互动、温柔但有流动感的段落。

可选动作：

- 沿主舞台或延伸台缓步行走，步伐从容。
- 手臂向外轻轻打开，邀请观众进入情绪。
- 单手做缓慢波浪、抚过空气、指向远处或指向观众。
- 转身回看 LED 屏或观众席。
- 走到乐手或伴唱附近，用点头、对视或侧身完成互动。
- 在强拍处抬手，再在弱拍处收回胸前。

### 高动态

用于副歌、高潮、drop、全场大合唱、情绪爆发或终章释放。

可选动作：

- 大步走向舞台中央或 T 台前端。
- 双臂张开或高举，动作落在强拍上。
- 仰头演唱，配合逆光和烟雾形成轮廓。
- 转身、半圈环绕或向观众区伸手。
- 与观众做呼应：举麦、挥手、带动合唱。
- 与舞者或乐队在同一节奏点完成统一动作。

## 表情与视线

- 闭眼：沉浸、内敛、抒情、情绪蓄力。
- 仰头：释放、渴望、高潮、仪式感。
- 低头：脆弱、回忆、独白、情绪收束。
- 直视镜头：亲密连接、对话感、直给情绪。
- 看向远方：叙事感、梦境感、回忆感。
- 微笑：轻松、亲和、治愈。
- 灿烂笑容：互动、明亮、可爱、结尾释放。

## 互动设计

互动对象可以是观众、乐手、伴唱、舞者、嘉宾或镜头。每次互动必须明确：谁看向谁、身体距离如何变化、动作落点是什么。

常用互动：

- 向观众伸手或举麦，让观众成为情绪放大器。
- 看向乐手或伴唱，形成现场音乐关系。
- 与另一位表演者对视、错身、并肩或面对面合唱。
- 鞠躬、挥手或点头，用于章节结束或感谢段落。

## 动作组合公式

### 抒情独唱

稳定站立或坐姿 → 胸前握麦 → 闭眼或低头 → 手按胸口 → 中景缓慢拉远。

### 情绪推进

中景站立 → 缓步走位 → 单手外展 → 视线看向观众 → 灯光增强。

### 高潮爆发

中景蓄力 → 面部特写 → 仰头或双臂张开 → 全景灯光全开 → 观众反应桥接。

### 互动收束

主唱动作完成 → 观众合唱或挥手 → 主唱微笑回应 → 镜头拉远到舞台整体。
