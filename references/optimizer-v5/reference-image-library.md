# Reference Image Library

This file is the text index for the internal realism reference library.

The image files are stored in:

```text
assets/reference-library/
```

## Purpose

Use this library to improve **AIMV / AI short-drama realism** for OpenAI image-family prompts, including GPT-image, image, image2, and future OpenAI image models.

The library is not a style pack. It is a realism benchmark that teaches the skill what believable camera-captured images should feel like:

- real skin under imperfect light
- real lens exposure limits
- natural motion softness
- unpolished background behavior
- believable wardrobe texture
- real crowd/stage/location scale
- practical lighting, LED spill, haze, reflections, rain, dust, sweat, and atmosphere
- short-drama / music-video continuity frames that look filmed rather than generated

## Non-negotiable rule

Never use reference-library images as identity sources. The user's uploaded face reference always overrides the library.

Use the library only for:
- camera realism
- lighting behavior
- material realism
- shot grammar
- anti-AI failure prevention
- scene-type examples

Do not copy or mention visible watermarks, creator names, platform marks, signatures, usernames, embedded text, filenames, titles, exact subjects, exact scenes, or exact compositions from library images. Borrow only the photographic principle and adapt it to the user's scene.

## Library usage workflow

When a prompt task is about AIMV, AI short drama, music video, concert stage, live event, sci-fi realism, emotional scene stills, or reference consistency:

1. Prefer the user's uploaded reference images for identity and scene specifics.
2. Use this library only as a realism benchmark.
3. Borrow the *principle*, not the content: lens behavior, light falloff, natural imperfection, exposure, motion, texture, and framing.
4. Keep all final prompt text aligned with the user's requested character and scene.
5. Do not mention library filenames in normal user-facing prompt output unless the user asks.

## Reference selection guide

- For interiors with sunlight, dust, smoke, or emotional stillness, consult refs 01, 02, 03, 09, 14, 15, 19.
- For outdoor natural light, sunset, forest, water, or warm backlight, consult refs 04, 05, 07, 08, 12, 16, 18.
- For concert, stage, singer, crowd, and live-event realism, consult refs 06, 13, 17.
- For sci-fi that must still look filmed, consult ref 11 and borrow only the physical armor, real lens, light spill, and material-contact logic.
- For short-drama portrait continuity and believable human framing, consult refs 07, 08, 10, 15, 16, 19.

- file: `assets/reference-library/ref-01-dusty-window-room.webp`
- category: interior / emotional scene / volumetric sunlight

- camera cues: Wide observational frame, low-detail silhouettes, imperfect exposure, large dark areas, no need to reveal every face or corner.

- use when: Short-drama loneliness, memory scenes, rehearsal-room mood, interior AIMV shots, characters separated by distance and light.
