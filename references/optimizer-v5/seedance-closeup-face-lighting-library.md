# Seedance 特写镜头面光提示词库（40 组）

适配版本：Seedance 2.0 / 2.1

## 设计标准

全部遵循「主光锚定 + 骨骼明暗 + 细节质感 + 环境反光」四层真实感结构，精准落到鼻梁、眼窝、下颌、瞳孔、颧骨、唇面和皮肤肌理，解决平光、塑料感、平面脸问题。

## 调用规则

- 单镜头仅选 1 组面光描述。
- 主光方向唯一，辅助反光不超过 2 组。
- 面光模块固定放在「主体 + 表情细节」之后、「场景环境 + 虚化」之前。
- 核心骨骼光影词可加权 1.1-1.2，最高不超过 1.3。
- 情绪向特写优先单侧光 / 强对比光；温柔日常优先柔侧光 / 漫射光；仙侠类优先自发光 / 环绕光。
