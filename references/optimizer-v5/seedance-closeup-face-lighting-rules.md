# 人物面光 / 骨骼光影强制书写规范

## 适用范围

所有 Seedance 古风写实类视频提示词生成 / 改造场景。

## 核心目的

从根源解决 AI 生成的面部平光、塑料磨皮感、五官扁平问题，锚定真实光影体积感，提升画面写实质感。

## 执行逻辑

满足触发条件时必须强制写入，不满足时可省略，禁止无意义堆砌。

---

# 模块一：人物面光 / 骨骼光影一致性强制规则

## 一、底层原则：所有面光必须有锚点，禁止凭空生成

人物面部光影绝对不能独立撰写，必须从「参考图光源」或「全局主光源」推导而来，做到 **"光源方向统一、冷暖统一、强度统一"**。

## 二、分场景锚定规则

### 场景 1：存在人物 / 场景参考图时（90% 常用场景）

必须先提取参考图的光源三要素，再对应推导人物面光，禁止与参考图光源冲突：

| 要素 | 提取内容 |
|------|----------|
| 光源方向 | 正面光 / 侧方光 / 侧逆光 / 逆光 / 顶光 |
| 光源冷暖 | 暖金夕阳 / 冷青天光 / 暖黄灯笼光 / 冷白月光 |
| 光源强度 | 柔和弥散 / 硬朗清晰 / 微弱朦胧 |

**标准锚定写法（格式参考，不得照搬内容）：**
> 面光完全匹配参考图的夕阳侧逆光逻辑，侧后方暖金光线勾勒轮廓，面部受环境光柔和补亮，明暗过渡与参考图保持一致。

### 场景 2：无参考图、纯文字创作时

必须先在「全局场景设定」中明确唯一主光源，后续所有人物面光、环境光影全部从该主光源推导，全程光源方向不允许突变。

- **错误**（示例说明原则，非模板）：开头写夕阳侧逆光，中段特写突然变成正面平光
- **正确**（示例说明原则，非模板）：全程主光源为侧方夕阳，近景特写依然是侧方光打亮半脸，另一侧暗部过渡

## 三、强制校验四步法（写完面光必须自查，不满足则重写）

| # | 校验维度 | 检查内容 |
|---|----------|----------|
| 1 | 方向校验 | 面光的来向，是否与全局主光源 / 参考图光源完全一致？ |
| 2 | 机位校验 | 光影点位是否匹配当前机位？（纯背影禁止写鼻梁、颧骨等面部正面高光） |
| 3 | 强度校验 | 光影软硬程度，是否和时间、天气匹配？（阴天薄雾不能写硬朗明暗交界线） |
| 4 | 景别校验 | 骨骼细节颗粒度是否匹配景别？（大远景不用写眼窝、唇峰等微观光影） |

## 四、分机位标准光影点位库（按需选用，禁止超范围乱写）

| 机位类型 | 可写高光点位 | 可写暗部点位 | 绝对禁止写 |
|----------|-------------|-------------|-----------|
| 正脸 / 四分之三侧脸 | 鼻梁、眉骨、颧骨最高点、唇峰、眼窝上缘 | 眼窝凹陷、鼻翼侧影、下颌角阴影、颈部阴影 | 后脑勺高光、肩背轮廓光 |
| 纯侧脸 | 鼻梁外缘、颧骨外缘、下颌线、耳郭边缘 | 侧脸凹陷、颈侧阴影 | 正面眉骨、唇峰高光 |
| 背影 / 背侧 | 发丝边缘、肩线、披帛边缘、发饰金属凸起 | 肩颈暗部层次 | 任何面部骨骼光影（鼻梁、颧骨、眼窝等） |
| 大远景 / 全景 | 人物整体轮廓光 | 人物暗部大关系 | 所有面部骨骼细节 |

# 模块二：强制必写触发条件 & 书写规范

## 一、强制必写触发条件（满足任意一条，必须补充骨骼光影描述）

| # | 触发维度 | 条件 |
|---|----------|------|
| 1 | 景别触发 | 镜头为中近景、近景、面部特写，人物面部占画面比例 ≥ 1/3，且面部为清晰对焦状态 |
| 2 | 机位触发 | 人物呈正脸、侧正脸、四分之三侧脸朝向镜头，面部五官完整可见 |
| 3 | 光源触发 | 场景使用侧光、侧逆光、逆光、黄昏斜阳、轮廓光、单点光源等非均匀平光环境 |
| 4 | 风格触发 | 整体风格要求为真实电影质感、写实摄影风、古风写实质感，对画面真实度有高要求 |
| 5 | 情绪触发 | 镜头为人物静态凝视、情绪表达、慢动作细节展示的重点情绪镜头 |

## 二、建议补充场景（满足可优先写入，提升画面质感）

- 中景镜头，人物为画面绝对视觉核心
- 光源方向随时间变化的渐变镜头
- 需要强化人物清冷 / 温柔 / 破碎感等特定气质的段落
- 人物从背光转向受光的动态镜头

## 三、标准书写规范（必须严格遵循，禁止空泛描述）

### 1. 固定书写结构

必须按「光源方向 → 骨骼高光点位 → 暗部过渡 → 整体质感」的顺序书写，所有描述必须落到具体可视的骨骼 / 结构点位，禁止抽象形容词。

### 2. 放置位置规则

紧跟在「人物动作 + 表情描述」之后、环境细节描述之前，属于人物属性模块，保证中高权重，不与环境光影混杂。

对应 7 项时间轴中的位置：
```
人物核心动作 → [人物光影（此处写入骨骼光影）] → 环境细节
```

### 3. 权重规则

- 核心光影关键词可加 1.1-1.15 权重（A 级质感强化），最高不超过 1.2
- 单段仅给 1 个核心词加权，禁止多处重复加权

# 模块三：分场景标准模板

> ⚠️ **以下模板仅用于参考格式结构和写作手法（光源方向→骨骼点位→暗部过渡→整体质感的四段顺序），严禁直接照搬文本内容。** 实际生成时必须根据当前场景的具体光源方向、机位、景别、人物身份和情绪重新撰写，只复用结构框架，完全替换具体内容。

### 模板 1：夕阳侧逆光・背侧机位（适配山顶、湖畔常用场景）

> 夕阳侧逆光从侧后方打来，发丝边缘、肩线、披帛与发饰金属凸起勾勒出淡金色柔和高光，肩颈形成自然明暗层次；面部仅侧缘受光，整体融入环境柔光，无突兀正面亮部。

### 模板 2：侧方柔光・四分之三侧脸（适配近景特写、情绪镜头）

> 侧方暖柔天光打亮半张侧脸，鼻梁、颧骨带温润高光，眼窝、鼻翼与下颌角有浅淡自然阴影，明暗过渡柔和细腻，皮肤保留真实肌理，无平光塑料感。

### 模板 3：正面弥散天光・正脸机位

> 柔和正面弥散天光铺脸，鼻梁与唇峰带细微高光，眼窝、鼻翼有极浅阴影过渡，面部明暗层次均匀自然，无强烈明暗交界，整体清透干净。

### 模板 4：黄昏侧逆光（通用，适配山顶 / 湖畔 / 亭台场景）

> 夕阳侧逆光从侧后方打来，鼻梁、眉骨、肩线与发梢边缘勾勒出淡金色柔和高光，下颌线与颈部形成自然明暗过渡，面部明暗层次柔和，无生硬边界。

### 模板 5：正面柔光（近景正面镜头）

> 柔和正面天光铺脸，鼻梁与唇峰带细微高光，眼窝与鼻翼侧有浅淡阴影，面部明暗过渡自然，皮肤保留真实肌理，无过度磨皮。

### 模板 6：侧方环境光（桥边、亭内侧面镜头）

> 侧方暖光打亮半张侧脸，颧骨与鼻梁带温润高光，另一侧面部融入柔和暗部，下颌线明暗交界清晰自然，面部体积感充足。

# 模块四：禁写条款（绝对禁止出现）

| # | 禁止项 | 原因 |
|---|--------|------|
| 1 | 空泛描述：如"光影立体、五官精致、面部有氛围感" | 无具体落点，写了等同于没写 |
| 2 | 光影矛盾：面光方向与全局环境光源 / 参考图光源方向不一致 | 环境侧逆光 + 面光正面光 = 穿帮 |
| 3 | 过度夸张：如"强烈高光、明暗对比强烈" | 古风写实证以柔和过渡为主，过强失真 |
| 4 | 远景加光影：大远景、全景、人物占比极小 | 浪费权重且无生效意义 |
| 5 | 背面加光影：纯背影、后脑勺镜头 | 面部不可见，仅保留肩线、发丝轮廓光 |
| 6 | 凭空生成：未从参考图或全局主光源推导，面光独立撰写 | 方向/冷暖/强度必然穿帮 |
| 7 | 跨机位乱写：背影镜头写鼻梁高光，侧脸写正面眉骨 | 违反机位逻辑 |
