# Command System

## Supported modes

- generate prompt
- optimize prompt
- audit prompt
- repair prompt
- translate image/video idea into prompt
- character sheet / three-view / multi-view
- Seedance expression prompt
- Seedance lighting prompt
- Seedance face lighting prompt
- Seedance camera movement prompt
- Seedance industrial storyboard prompt
- workflow/rule update
- GitHub sync

## Rules

- Prompt commands produce copy-ready output.
- Rule-update commands modify documentation/rules, not previous prompts.
- Seedance complete outputs separate positive main prompt and compact negative prompt.
- Avoid generic style tags and model-incompatible syntax.
- Use simple weights only when they strengthen crucial details.
