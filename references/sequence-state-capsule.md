# Sequence State Capsule 连续片段状态胶囊

## 目的

连续片段状态胶囊用于多段视频、连续剧情、系列 MV、AI 短剧、分章节故事板、首帧尾帧续写和“接着上一段继续”的任务。

它解决的问题：

- 上一段已经发生的动作被重复。
- 当前段提前出现未来剧情。
- 人物、服装、道具、场景、光向、风向、情绪断裂。
- 第二段、第三段和上一段结尾接不上。
- 多段视频缺少稳定的开始状态和结束状态。

## 启动条件

只要用户出现以下任一表达，就启动本规则：

```text
继续上一段
接着上一条
做下一段
第二段 / 第三段 / 第N段
系列视频
连续剧情
多段视频
多段 MV
章节式分镜
上一段结尾接下一段开头
保持上一段人物和场景继续发展
从这个尾帧继续
根据上一条的结尾继续生成
```

## 状态机挂接位置

本规则在以下阶段启动：

```text
S3_PROJECT_CLASSIFICATION
S6_BRANCH_CHECK
S8_FINAL_GENERATION
S9_INTERNAL_QA
```

- `S3_PROJECT_CLASSIFICATION`：判断是否属于连续项目。
- `S6_BRANCH_CHECK`：触发连续片段分支。
- `S8_FINAL_GENERATION`：生成当前片段提示词时，只写当前片段，不泄露未来段。
- `S9_INTERNAL_QA`：检查是否重演旧动作、提前泄露未来事件、破坏连续性。

## 状态胶囊字段

连续项目必须内部建立以下状态胶囊。用户不要求时，不必完整展示给用户。

```text
【连续片段状态胶囊】
project_id：
clip_id：
parent_clip_id：

already_happened：
上一段已经发生的事件、动作、情绪变化、道具变化、空间位置变化。

current_clip_only：
当前片段只允许发生的事件、动作、情绪推进和镜头变化。

reserved_for_later：
保留给未来片段的事件，不得写入当前最终提示词。

planned_start_state：
当前片段开始时的人物、姿态、位置、道具、场景、光线、风向、情绪状态。

planned_end_state：
当前片段结束时必须抵达的状态，用于下一段衔接。

continuity_locks：
必须保持连续的角色身份、脸、发型、服装、道具、空间关系、光源方向、风向、情绪基调、镜头轴线。

do_not_repeat：
上一段已经完成、当前段不得重演的动作或镜头。

do_not_reveal_yet：
未来片段才出现、当前段不得提前出现的信息。
```

## 当前段生成原则

生成连续片段时，最终提示词只能写当前片段。

必须遵守：

1. 不重演上一段已经完成的动作。
2. 不提前写未来片段的事件。
3. 当前段开始状态必须接得上上一段结束状态。
4. 当前段结束状态必须为下一段留下清晰接口。
5. 角色、服装、道具、场景、光源、风向、情绪必须连续。
6. 如果有首帧 / 尾帧，首帧控制当前段起点，尾帧控制当前段终点。
7. 未来剧情只能出现在规划说明或内部胶囊中，不得进入当前最终提示词。

## 正确写法

用户说：

```text
继续上一段，做第二段，女主已经走进亭子，这一段让她开始弹古筝，最后抬头看向镜头。
```

内部状态胶囊：

```text
already_happened：女主已经从桥上走入亭子，停在古筝前，风从右前方吹向左后方。
current_clip_only：女主坐下，手指触弦，开始弹奏，情绪从平静转为克制低落，结尾抬眼看向镜头。
reserved_for_later：起身离开亭子、落泪、回眸、远景离场。
planned_start_state：女主站在亭内古筝前，服装和发型延续上一段，山风方向不变。
planned_end_state：女主坐在古筝前，指尖停在琴弦上，抬眼看向镜头。
do_not_repeat：不要重复从桥上走入亭子的动作。
do_not_reveal_yet：不要写起身离开、落泪和远景离场。
```

最终提示词只写当前段：

```text
这一段从女主站在亭内古筝前开始，不重复她走入亭子的过程。她缓缓坐下，右手指尖轻触琴弦，左手压弦，情绪从平静转为克制低落，结尾在琴声余韵中缓慢抬眼看向镜头。
```

## 错误写法

错误：

```text
她再次从桥上走入亭子，随后坐下弹琴，最后起身离开亭子并落泪。
```

原因：

```text
重复上一段动作：再次从桥上走入亭子。
提前泄露未来事件：起身离开亭子并落泪。
当前段事件过多，破坏连续片段控制。
```

## 与人物镜头强制覆盖规则的关系

本规则只管理连续片段状态，不替代人物镜头强制覆盖规则。

如果当前片段中出现人物，仍必须按人物镜头强制覆盖规则执行：

- 人物动作必写。
- 表情与肢体联动必写。
- 说话 / 唱歌 / 对口型时写口型。
- 中近景 / 近景 / 特写写人物面部光影。
- 有动态材质写风向 / 柔体动力学。
- 有参考图锁定人物一致性。

## 负面提示词建议

连续片段项目的负面提示词可加入：

```text
连续性类：重复上一段已完成动作、未来事件提前出现、人物位置跳变、道具状态不连续、情绪断裂、场景空间关系断裂、光源方向跳变、风向跳变、上一段结尾无法衔接
```
