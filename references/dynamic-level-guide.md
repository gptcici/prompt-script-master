# 动态级别判断指南

动态级别用于在复述阶段判断视频应采用多大的动作幅度、镜头速度、景别变化频率和灯光变化强度。它属于确认前用户沟通层，必须等用户确认后才能写入最终提示词。

> **流程衔接**：本指南负责"自动建议"，确认动作由 `confirmation-gate.md` 接管。流程为：自动分析→建议输出→等待用户确认→确认后才进入下一阶段。

## 轻动态

适合：

- 对口型唱歌
- 抒情演绎
- 慢歌
- 人像情绪
- 轻微律动
- 微表情与眼神变化
- 首尾帧轻过渡

动作特征：动作小、速度慢、身体重心稳定，更多依靠眼神、头部、手部和呼吸变化。

镜头特征：固定、慢推、轻微横移、慢拉焦、近景和特写较多。

## 中动态

适合：

- 略有动态的流行音乐
- 广告 MV
- 轻舞台
- 商业展示
- 轻量卡点
- 产品广告
- 人物从克制到释放的情绪推进

动作特征：允许走位、手臂打开、身体左右律动、轻卡点、转身、向前一步定格。

镜头特征：推镜、横移、轻环绕、升降、拉焦、景别从中景到半身或近景递进。

## 高动态

适合：

- 演唱会级别
- 摇滚
- 电音
- 舞曲
- drop
- 强拍密集副歌
- 团舞
- 舞台高光
- 动作高潮

动作特征：动作幅度大、速度快、强拍明显，可包含快速走位、跳跃、强卡点、团舞队形、强烈手臂动作。

镜头特征：快速推近、快速拉远、鞭甩摇镜、甩变焦、低角度环绕、螺旋上升、强光爆闪、景别切换更频繁。

## 判断输出格式

复述阶段必须写：

```text
我建议动态级别为：轻 / 中 / 高
判断依据：……
这个级别会影响：动作幅度、运镜速度、景别切换频率、灯光变化强度。
需要你确认：是否采用这个动态级别？
```

如果音乐风格、节奏或参考资料不足，必须写成临时判断，不得直接设计确定动作。
