# MV 叙事结构库

用于 MV、AIMV、音乐短片、歌词驱动短片、舞台与多场景穿插项目。目标是让 MV 有情绪结构，而不是只有漂亮镜头堆叠。

## 基础结构：现实锚点 + 幻想分支

适用于爱情、回忆、失恋、成长、内心独白、喜剧或群像类 MV。

结构：

1. 现实锚点：一个稳定空间承载主角真实情绪，例如舞台、房间、车内、天台、走廊、录音室。
2. 分支场景：多个副空间承载不同情绪侧面，例如浪漫、愤怒、遗憾、逃离、诱惑、孤独、释放。
3. 反复切换：主空间负责回收情绪，副空间负责展开隐喻。
4. 情绪崩塌：副空间变得更夸张、更快、更失控。
5. 统一收束：所有空间通过同一动作、节奏、灯光或剪辑逻辑合并。

## 五幕式 MV 结构

### 第一幕：引入

建立主空间和主角状态。动作少，镜头稳，光影明确。让观众知道主角在什么情绪里。

### 第二幕：展开

引入多个空间、人物关系或情绪对照。剪辑开始变快，但每个空间都要有清晰符号：一个动作、一个物件、一个光影状态或一个人物反应。

### 第三幕：转折

浪漫、幻想或秩序开始崩塌。人物动作变夸张，光影对比增强，切镜频率上升。

### 第四幕：高潮

回到主空间或核心关系，情绪集中爆发。使用推近、环绕、强特写、逆光、烟雾或快速蒙太奇。

### 第五幕：释放

所有分支空间不再互相割裂，而是在统一节奏下合并。可用齐舞、同步动作、同一歌词动作、同一光色变化或多空间混剪收束。

## 空间调度原则

- 主空间是情绪圆心，不需要太多变化，但必须多次回到这里。
- 副空间是情绪投射，不要超过用户项目能承载的数量。
- 每个副空间必须有一个明确功能：回忆、诱惑、逃离、愤怒、喜剧、失控、自由、告别。
- 副空间之间的切换要跟音乐节拍或歌词情绪相关，不要随机切。
- 结尾最好用一个统一动作或视觉规则把空间收回。

## 喜剧型 MV 结构

适合轻快、复古、爱情喜剧、反差、群像闹剧。

循环：

角色进入 → 做出夸张求爱 / 表演 / 失败动作 → 主角冷静旁观或轻微反应 → 切到下一空间。

结尾：

所有角色被音乐统一，进入同一动作节奏或群体释放。

## 抒情型 MV 结构

适合失恋、回忆、温柔、治愈、孤独。

循环：

主空间独处 → 回忆空间闪回 → 主角表情变化 → 回到主空间 → 光线或动作更内敛。

结尾：

镜头拉远、光线变弱、主角放下动作、空间空出来。

## 高能舞台型 MV 结构

适合副歌爆发、唱跳、演唱会、群舞。

循环：

舞台中心 → 动作细节 → 观众反应 → 舞台全景 → 主角强拍动作。

结尾：

群体动作、灯光全开、拉远到空间规模或定格主角姿态。

## 禁止项

- 不要复刻任何具体参考 MV 的人物、道具组合、场景组合或桥段顺序。
- 不要在最终脚本中出现参考 MV 名称、歌手名或原始时间戳。
- 不要把“多个场景快速切换”写成无逻辑混剪，必须说明每个空间的情绪功能。
