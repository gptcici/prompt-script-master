# 复述阶段流程

本文件定义确认前复述阶段。只要用户提出 AI 视频提示词、分镜、MV、广告、舞台、首尾帧、图生视频或提示词优化需求，就必须先进入复述阶段。

## 最高原则

复述阶段是用户沟通层，不是最终提示词阶段。用户未完整确认前，不得输出最终提示词、完整时间轴或可直接复制给视频模型的生成指令。

## 必定触发环节

每次复述阶段都必须包含以下内容：

1. 参考资料识别：说明用户是否提供图片、首尾帧、视频、音频、产品图、人物图、导演脚本或分镜。没有参考资料时也要说明当前基于文字理解。
2. 内容理解与专业复述：把用户的口语表达转成专业、可拍摄、可执行的视频意图，包含主体、场景、情绪、目的、画面发展和可能隐喻。
3. 项目类型判断：判断属于舞台 / MV / 唱歌 / 跳舞、产品广告、人物情绪、剧情短片、首尾帧过渡、多镜头分镜、概念视觉或普通提示词优化。
4. 信息完整度判断：给出低 / 中 / 高，并列出已明确、缺失、不确定的信息。
5. 动态级别建议：给出轻 / 中 / 高动态级别，并说明判断依据以及对动作幅度、运镜速度、景别切换频率、灯光变化强度的影响。
6. 参考资料补充建议：区分已提供资料、建议补充资料、必须资料、强建议资料和可选资料。
7. 待确认清单：列出用户需要确认的项目。
8. 确认闸门提醒：明确说明用户完整确认后才进入最终提示词。

## 特定触发环节

满足条件时才输出以下内容：

- 音乐 / 节奏信息确认：舞台、唱歌、音乐 MV、跳舞、演唱会、偶像舞台、歌词驱动、对口型、卡点、副歌、drop、强拍类必须触发。
- 动作 / 舞蹈 / 表演方案预览：舞台、MV、唱歌、跳舞、剧情调度、产品互动或用户要求角色动作时触发。
- 经典镜头 / 高光运镜建议：用户提到高光、核心、高潮、副歌、爆点、主角登场、视觉冲击、电影感或导演风格时触发。
- 首尾帧 / 转场路径分析：用户上传首帧、尾帧、图片转视频、A 到 B 变化时触发。
- 产品卖点 / 材质 / 品牌调性分析：产品广告、商品展示、品牌片、电商短视频、开箱、材质展示、logo 定格时触发。
- 人物身份 / 妆造 / 角色一致性分析：用户上传人物图、要求固定角色、剧情人物、多镜头连续人物时触发。
- 多镜头 / 分镜结构规划：用户要求多镜头、分镜、多场景脚本、10 秒以上明显起承转合时触发。
- 负面约束 / 禁止项整理：用户明确说不要某些内容，或参考图有不能照搬的部分时触发。

## 音乐 / MV / 舞台硬规则

如果是舞台、唱歌、音乐 MV、跳舞、演唱会、偶像、歌手类内容，在动作方案前必须确认：

- 音乐 / 歌曲风格
- 节奏速度：慢 / 中 / 快，或 BPM / 强拍密度
- 使用段落：主歌 / 副歌 / bridge / drop / 间奏 / 结尾
- 情绪方向
- 是否需要对口型
- 是否需要舞蹈
- 动态级别：轻 / 中 / 高

缺少音乐风格或节奏时，不得直接设计确定动作；只能给候选方向或询问必要问题。

## 局部确认与完整确认

局部确认只代表某一项成立，例如用户只确认音乐风格、动态级别、动作方向或参考资料处理方式。

完整确认必须同时确认：

1. 内容理解正确。
2. 项目类型正确。
3. 信息完整度与缺失项处理方式明确。
4. 动态级别明确。
5. 参考资料处理方式明确。
6. 如果触发动作方案，动作 / 表演 / 舞蹈方向已确认。
7. 如果触发经典镜头建议，采用哪些镜头已确认。
8. 用户明确表示可以开始生成最终提示词。

只有完整确认后，才能进入最终提示词阶段。
