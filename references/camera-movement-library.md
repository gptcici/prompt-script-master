# 基础运镜库

基础运镜库是经典镜头库的底层动作语法。最终提示词必须写具体运镜，不得只写运镜名称。

## 基础运镜

- 固定镜头：摄影机固定不动，主体在画面中完成动作。
- 推镜：摄影机从中景缓慢向主体推进，景别从中景进入近景，焦点锁定主体。
- 拉镜：摄影机从近景后退，逐渐露出完整环境和空间关系。
- 横移：摄影机向左或向右平稳移动，主体保持画面中心或从一侧进入画面。
- 升镜：摄影机从低位上升，空间逐渐打开。
- 降镜：摄影机从高处下降，逐渐贴近主体或落到视线高度。
- 俯仰摇镜：摄影机原地向上或向下倾斜。
- 左右摇镜：摄影机原地向左或向右转动。
- 变焦：摄影机不移动，通过焦距变化拉近或拉远主体。
- 快速推近：快速靠近主体，制造压迫感或冲击。
- 快速拉远：快速远离主体，突出主体与环境关系。
- 手持抖动：模拟真实手持拍摄和现场感。
- 跟拍：摄影机与主体保持相近速度持续跟随。
- 追拍：摄影机从主体后方跟随移动。
- 过肩镜：越过 A 角色肩膀拍摄 B 角色或目标。
- 主观镜头：摄影机模拟角色视角移动。
- 低角度仰拍：增强力量感、压迫感和英雄感。
- 高角度俯拍：突出空间布局或主体弱小 / 孤立。
- 鸟瞰顶拍：从正上方垂直拍摄，突出路径、队形、空间图案。
- 无人机飞行：像无人机一样穿越空间，但画面中不要出现无人机本体。
- 环绕镜头：摄影机围绕主体做圆弧运动，主体保持中心。
- 旋转镜：摄影机围绕自身光轴旋转。
- 螺旋上升 / 下降：一边环绕主体一边升降。
- 穿越前景：摄影机从遮挡物后穿出，主体逐渐显露。
- 拉焦 / 转焦：焦点从前景转移到主体，或从主体转移到背景目标。
- 甩镜：快速甩向另一个目标，产生短暂运动模糊后落到新主体。
- 推拉变焦：摄影机移动与焦距反向变化，制造背景透视异常变化。

## 组合规则

单个时间段最多组合 2-3 个主要运镜动作，必须区分主动作和辅助动作。不要在同一段内同时堆叠推进、环绕、升降、旋转、变焦、转焦和穿越前景，除非拆成多个时间段。
