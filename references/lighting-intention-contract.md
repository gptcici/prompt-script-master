
# Lighting Intention Contract 光影意图合约

## 目的

本规则用于把“氛围、电影感、情绪、质感、舞台光、月光、烛光、室内光、产品光、雨夜光、古风光影”等要求转化为物理可见的灯光结构。它增强全局光源锚点和人物面部光影规则，但不替代它们。

## 启动条件

出现以下任一情况时读取本规则：

```text
光影、灯光、面光、月光、烛光、夕阳、灯笼、舞台光、逆光、侧光、轮廓光、电影感、氛围感、情绪、孤独、压迫、温暖、冷峻、梦幻、产品光、材质反光、阴影、色温、雾气、雨夜、烟尘、HDR、调色、ACES
```

## 光影合约字段

关键镜头必须明确：

```text
主光源：太阳、月光、窗光、灯笼、烛火、台灯、舞台灯、霓虹、屏幕、闪电等。
方向：前侧、侧后、顶侧、窗外、画面左/右、低角度/高角度。
色温：暖金、烛黄、冷蓝、青绿荧光、钠灯橙、阴天中性等。
光质：硬光、柔光、散射、窄条光、跳光、反射光。
阴影：眼窝、下颌、衣褶、建筑投影、地面阴影、暗部细节。
反射：皮肤、金属、玉石、丝绸、水面、玻璃、湿地面的反光。
空气介质：雾、烟、尘、雨丝、雪、香火、蒸汽如何被光捕捉。
变化：光是否随门开、风吹、闪电、舞台节拍、人物移动发生变化。
```

## 光源选择规则

根据场景选择能被观众理解的光源：

| 场景 | 推荐光源 | 用法 |
|---|---|---|
| 室内亲密 | 烛火、台灯、窗光、灯笼 | 光源可见或有动机，适合情绪近景 |
| 古风夜景 | 冷月光 + 暖灯笼 / 烛火 | 冷暖分层，人物面部有眼神光 |
| 山水仙侠 | 低角度夕阳、云海反光、冷月轮廓 | 强化轮廓、披帛、发丝、云雾层次 |
| 舞台 / MV | 主舞台灯、追光、背光、LED 屏反光 | 光随节拍变化，但主光逻辑稳定 |
| 产品 / 材质 | 窄条光、柔箱、反射板、边缘光 | 证明材质、边缘和结构，不乱动产品 |
| 雨夜 / 街景 | 霓虹、车灯、路灯、湿地反射 | 让雨丝和地面反光承载氛围 |
| 压迫 / 对峙 | 侧硬光、低照度、局部面光 | 阴影边界和空间压迫服务情绪 |

## 情绪转光影规则

不要只写“孤独、温柔、压迫”。要把情绪翻译成光影：

```text
孤独：大面积暗部 + 远处小光源 + 人物脸部保留一条弱眼神光。
温柔：大面积柔光 + 暗部低对比 + 皮肤和布料有轻微暖色反射。
压迫：顶侧硬光或窄侧光 + 背景压暗 + 下颌和眼窝阴影更重。
决心：推近时主光在眼神光中稳定出现，背景光降低。
梦幻：逆光穿过雾、纱、发丝和云气，但人物脸部仍有清楚结构。
```

## 人物面光强化

当人物中近景、近景、特写出现时，必须和《人物镜头强制覆盖规则》联动：

```text
主光方向 + 光质色温 + 鼻梁高光 + 眼窝明暗 + 颧骨亮面 + 下颌阴影 + 眼神光 + 皮肤微细节 + 环境弱反光
```

避免：

```text
面光柔和、电影感面光、全脸打亮、高级光影、真实光影
```

除非这些词后面有具体光源、方向、阴影和反射。

## 光与动态介质

当画面包含雾、烟、雨、雪、灰尘、香火、水面、玻璃、金属、丝绸、火焰或粒子时，必须联动 `physical-dynamics-standard.md`：

- 光束必须被介质显形，而不是凭空出现。
- 雾和烟应有来源、扩散、遮挡、边缘消散和受光层次。
- 雨雪应有方向、密度、遮挡关系和与光源的短暂高光。
- 水面应有接触点、涟漪、反射破碎和重新合拢。
- 金属、玉石、丝绸、湿地面应通过运动产生高光滑移或反射变化。

光影合约负责“光为什么存在、从哪里来”；物理动力学标准负责“动态介质如何接住、散射、遮挡或反射这束光”。

## 灯光变化规则

灯光变化必须有原因：

- 门缝打开，光束变宽。
- 云遮月，面部暗部加深。
- 舞台灯随副歌增强。
- 烛火被风压弯，人物脸上暖光抖动。
- 闪电短暂硬化轮廓。

避免无原因的“灯光突然变化”。

## 与色彩 / 后期的边界

提示词只能描述可见光、色彩、材质和情绪，不能保证真正的 ACES、HDR、LUT、CDL 或交付色彩空间。若用户要求专业调色交付，应输出两层：

1. 生成提示词中的可见光影目标。
2. 后期交付备注：由剪辑 / 调色流程验证色彩空间、肤色、产品色、HDR/SDR 或社媒版本。

## 负面提示词建议

```text
光影类：主光源不明、光源方向跳变、面部平光、眼窝死黑、下颌无结构、无眼神光、皮肤塑料反光、材质无反射、雾气不吃光、灯光变化无原因、冷暖光混乱
```
