# Reference Intake Self Check 参考图上传后统一自检流程（V0.9.82）

## 目的

本规则是所有上传参考图后的统一入口。它把“参考图是否清晰”“人脸是否健康”“哪些内容可继承”“哪些内容必须补写”“是否建议用户重新生成参考图”合并成一个固定流程，避免 I2V 极简原则在低质量参考图上误用。

本规则不替代 `reference-transfer-contract.md`，而是在分配参考图职责之前先判断每张图片是否值得作为可靠参考。

## 启动条件

只要用户上传任何图片，或请求图生视频、首尾帧、锁脸、换装、多参考图融合、角色图、三视图、海报、图片重绘、关键帧、构图参考、服装参考、发饰参考、场景参考，就先执行本规则。

## 固定执行顺序

```text
S2_REFERENCE_SCAN
  ↓
参考图上传后统一自检：识别每张图的类型、用途、关键区域和可读性
  ↓
reference-visibility-gate.md：判断人脸、服装、发型/发饰、背景、构图、首尾帧连接是否清晰
  ↓
face-integrity-check.md：只要有人脸，就检查生理结构、肤色连续、面部阴影和异常色块
  ↓
参考图可用性决策：可直接用 / 部分可用需补写 / 异常需重建 / 建议重新生成参考图
  ↓
reference-transfer-contract.md：只把清晰可信的信息分配为参考图职责
  ↓
output-length-mode-standard.md：再决定 I2V / R2V / FLF2V / IMAGE 的输出格式和长度
```

## 参考图自检维度

每张上传图片都必须内部判断以下项目：

1. **主体可识别**：主体是否清楚，是否被遮挡、裁切、低分辨率、运动模糊、过曝、过暗或压缩噪点破坏。
2. **人脸完整性**：脸型、五官、眼神、鼻唇、下颌、肤色、阴影是否符合生理逻辑；是否存在畸形、黑斑、异常色块、死黑阴影或塑料/蜡像皮肤。
3. **服装与发饰可读性**：服装轮廓、领口、袖型、裙摆、材质、纹样、发型、发饰、发带、流苏是否可辨。
4. **场景与构图可读性**：空间类型、前中后景、光源方向、主体位置、画幅、景别、运动空间是否明确。
5. **职责匹配性**：图片是否适合承担用户指定的职责，例如“锁脸图”必须脸清楚，“服装图”必须服装完整，“场景图”必须空间清楚。
6. **首尾帧可连接性**：首帧和尾帧主体、服装、场景、光源和动作路径是否能自然连接。

## 四档参考图可用性决策

### A. 可直接使用

关键区域清晰，画面没有明显生理或光影异常。可以执行 I2V 极简原则：图片已有内容不重复长篇重写，只写动作、运镜、光线变化、动力学和保持约束。

### B. 部分可用，必须补写

图片有一部分清楚、一部分不清楚。清楚部分可以作为参考职责；不清楚部分必须在提示词中补写。

例：脸清楚但服装下半身被裁切，则脸可用于 identity，服装必须补写结构、材质、颜色和裙摆/裤型。

### C. 异常可参考，但不得继承异常

图片有可用方向，但存在人脸畸形、异常色块、黑斑、死黑阴影、皮肤伪影、背景脏块或压缩噪点。只能继承抽象方向，不继承异常细节；必须在正文中重建自然结构，在负面提示词中写入异常故障项。

### D. 建议重新生成参考图

当不清晰或异常区域较多，导致图片无法可靠承担用户指定职责时，必须在确认阶段建议用户先重新生成或上传更清晰的参考图。

## 建议重新生成参考图的触发阈值

满足任意一条，即建议重新生成参考图，而不是硬套 I2V 极简原则：

- 关键主体不清楚，且用户要求锁脸、换装、三视图、角色连续或首尾帧精确过渡。
- 人脸、服装、发型、背景、构图这五类中有三类及以上不清晰。
- 用户要求锁脸，但脸部五官不可辨、被遮挡、过暗、过曝、低清或存在明显畸形/异常色块。
- 用户要求服装参考，但服装主体被裁切、遮挡或关键结构不可辨。
- 用户要求场景/构图参考，但背景大面积死黑、过曝、模糊或主体位置不可判断。
- 首帧和尾帧主体不一致、动作路径不可连接、光源/构图冲突严重。
- 参考图存在大面积 AI 伪影、异常黑斑、脏污色块、脸部死黑、压缩噪点或不符合生理逻辑的皮肤质感。

## 用户可见提示规则

普通确认模式下，如果建议重新生成参考图，应该直接告诉用户原因，并给出更适合的参考图要求。

写法示例：

```text
这张参考图中人物脸部和服装关键区域都不够清晰，如果继续使用，模型可能无法稳定锁脸和还原服装。建议先重新生成/上传一张更清晰的参考图：人物正面或三分之二侧脸，五官清楚，服装完整可见，光线不要死黑或过曝。
```

快捷生成模式下，不强制中断，但必须在提示词中补写不清晰区域，并排除异常区域；如果严重不可用，仍应说明“该参考图不适合承担锁脸/服装/首尾帧职责”。

## 与 I2V 极简原则的关系

I2V 极简原则必须排在本自检之后：

```text
只有清晰可信的图片内容才能省略；
不清晰内容必须补写；
异常内容必须排除和重建；
大面积不清晰时，建议重新生成参考图。
```

## 禁止行为

- 用户上传图片后，直接跳到 I2V 极简提示词，不做参考图可用性判断。
- 图片多处模糊仍只写“保持图中人物/服装/背景”。
- 锁脸任务中，人脸不清楚仍只写“锁脸”。
- 服装参考不清楚仍把服装职责完全交给图片。
- 人脸黑斑、异常阴影、死黑或色块被当作身份特征继承。
- 参考图严重不可用时不提醒用户重新生成。
