# Physical Dynamics Standard 物理动力学总规则

## Status
Version: V0.9.77

本规则用于把“动起来、风感、质感、烟雾、雨雪、水面、碎片、道具运动、接触、碰撞、重力、惯性、消散、环境互动”等要求转化为可被视频模型执行的物理动力学。它是 `seedance-wind-softbody-standard.md` 的上位扩展：风 / 柔体仍然保留，但不再是唯一动力学入口。

## 核心原则

动力学不是只写“有风、头发飘、衣服动”。任何动态都必须有：

```text
动力来源 → 作用对象 → 运动路径 → 材质响应 → 可见结果 → 结束状态 / 消散状态
```

如果画面中出现可动元素，必须判断它属于哪一类动力学，再决定是否写入正文提示词。

## 启动条件

出现以下任一内容时读取本规则：

```text
风、头发、发丝、衣服、披帛、袖口、裙摆、纱幔、旗帜、窗帘、云雾、烟、雨、雪、灰尘、花瓣、火焰、烛火、水面、浪花、涟漪、倒影、液体、碎片、沙尘、粒子、纸张、门、窗、杯子、琴弦、武器、饰品、珠链、流苏、动物、群体、车辆、道具运动、碰撞、落下、滑动、滚动、漂浮、爆发、消散、轨迹、惯性、重力、反作用、接触点
```

## 动力学分类

### 1. 风场动力学

用于发丝、衣料、披帛、纱幔、竹叶、云雾、雨线、烟尘、水面等受风影响的元素。

必须写清：

```text
风向、风力、连续性、脉冲变化、近景与远景是否同向、不同材质的响应速度差
```

### 2. 柔体动力学

用于头发、衣物、披帛、宽袖、裙摆、纱幔、旗帜、丝带、窗帘、布料、软皮革等。

必须写清：

```text
固定点、受力方向、拉伸、翻卷、折叠、回弹、重量、下坠、延迟、边缘与主体响应差
```

### 3. 流体动力学

用于水、雨、河流、酒水、茶水、泪水、墨汁、雾气、烟雾、蒸汽、云海等。

必须写清：

```text
来源、流向、速度、扩散、分流、回卷、涟漪、反射、消散、与地形/身体/道具的接触反应
```

### 4. 粒子 / 碎片 / VFX 动力学

用于灰尘、雪、花瓣、火星、纸屑、碎片、沙尘、香火、雨点、光粒、能量粒子。

必须写清：

```text
生成来源、密度、大小差、运动方向、速度差、受风影响、遮挡关系、受光方式、落点或消散方式
```

### 5. 刚体 / 道具动力学

用于门、杯子、琴、剑、伞、书、纸张、桌椅、车辆、产品、武器、饰品主体等硬质物体。

必须写清：

```text
接触点、支点、转轴、滑动/滚动/摆动路径、摩擦、碰撞、惯性、停止位置、是否影响周围物体
```

### 6. 接触 / 碰撞 / 反作用

用于手碰物体、脚踩水、衣袖扫过桌面、剑碰鞘、杯子放下、角色撞开门、雨点击打地面等。

必须写清：

```text
接触前状态、接触点、受力方向、反作用结果、声音/水纹/震动/位移、最终状态
```

### 7. 重力 / 惯性 / 延迟

用于下落、奔跑停顿、转身、布料回落、发尾延迟、饰品摆动、烟雾上升、水滴坠落。

必须写清：

```text
初始速度、惯性方向、延迟响应、重力下坠、回弹幅度、最终静止或继续运动状态
```

### 8. 光与介质互动

用于雾、烟、雨、雪、灰尘、水面、玻璃、金属、丝绸等承接光线的动态材质。

必须写清：

```text
光从哪里穿过/反射/散射，动态介质如何显出光束、边缘高光、暗部层次、遮挡和透明度变化
```

## 动力学锚点写法

复杂视频建议把原来的“全局风向 / 柔体动力学锚点”升级为：

```text
全局物理动力学锚点：本片段的主要动力来源为 [风 / 人物动作 / 道具接触 / 雨雪 / 水流 / 烟雾 / 舞台机械 / 爆发粒子]。[方向/力量/节奏] 全程保持一致或按 [剧情/节拍] 有明确变化。发丝、衣料、饰品、纱幔、云雾、水面、雨线、烟尘、道具分别按材质重量、固定点、受力路径、速度差、延迟、回弹、重力下坠、接触反应和消散状态表现，不同层级不能同速漂浮或无原因乱动。
```

如果画面只有风和柔体，可继续使用：

```text
全局风向 / 柔体动力学锚点：...
```

如果画面涉及雨、水、烟、火、粒子、道具接触或碰撞，必须使用更宽的“全局物理动力学锚点”。

## 写作顺序

复杂动态镜头优先按以下顺序写：

1. 动力来源：风、人物动作、重力、碰撞、机械、雨水、烟火、舞台效果。
2. 主要受力对象：人物、衣物、道具、环境、流体、粒子。
3. 运动路径：向哪里移动、绕哪里旋转、在哪里扩散、在哪里回卷。
4. 材质响应：软硬、轻重、湿干、透明、不透明、金属、丝绸、水面、烟雾。
5. 速度差：近景/远景、轻/重、根部/末端、主体/边缘。
6. 接触与反作用：碰到什么、造成什么位移、涟漪、震动、声音或光影变化。
7. 结束状态：停住、回落、消散、漂出画面、形成新的构图状态。

## 分层规则

### 人物层

- 身体动作产生衣物惯性。
- 头部转动带动发尾延迟。
- 手触碰道具必须产生接触结果。
- 情绪动作可通过指尖、肩颈、呼吸、衣袖、饰品小幅响应表达。

### 服装 / 饰品层

- 轻纱、披帛、丝带：响应快，幅度大，可拉伸、翻卷、回落。
- 厚裙、外袍、腰封：响应慢，幅度小，主要在边缘和下摆运动。
- 珠链、流苏、耳坠、发饰：小幅延迟摆动，不应像布条一样大幅飞舞。
- 金属饰品：反光变化比形变更明显。

### 道具层

- 琴弦：拨动后产生细小震动和高光闪烁，随后余震渐弱。
- 纸张：边缘先受力，再翻折或滑动，最后停在具体位置。
- 门窗：围绕转轴运动，带动气流、光束、烟线或帘幕变化。
- 杯盏 / 水面：接触后出现晃动、涟漪、反光偏移。
- 武器：有重量、惯性、停顿和反作用，不应无重力漂浮。

### 环境层

- 云雾：速度慢于发丝和衣物，可分流、上升、贴地流动或绕开建筑。
- 烟：受热上升，遇风偏移，边缘扩散，逐渐消散。
- 雨雪：有方向、密度、落点和遮挡关系，可被风斜吹。
- 水：受脚步、雨点、道具落入影响产生局部涟漪，不应全屏同速波动。
- 火焰 / 烛火：受风压弯、变亮或闪烁，同时影响周围暖光。

## 正确例子

### 古风人物 + 风 + 布料 + 云雾

```text
全局物理动力学锚点：山风从画面右前方持续吹向左后方，风向全程统一。女主耳侧碎发和发带最先响应，发尾产生轻微延迟；披帛向左后方拉伸后翻卷回落，厚重裙摆只在边缘缓慢摆动。亭外竹叶沿同一方向细密抖动，远处云雾速度更慢，贴着山壁上升后向左后方分流，与近景风动保持统一空间逻辑。
```

### 雨夜 + 水面 + 光影

```text
全局物理动力学锚点：斜雨从画面左上落向右下，雨线穿过蓝色路灯形成短暂高光。人物脚步落入浅水坑时，接触点向外扩散细小涟漪，湿地面霓虹倒影被涟漪拉碎后重新合拢。远处雨幕密度更高、速度更慢，背景车灯被雨雾散射成柔软光斑。
```

### 道具接触 + 烛火 + 烟线

```text
全局物理动力学锚点：木门被人物左手缓慢推开，门轴围绕右侧铰链转动，门缝气流先压弯烛火，再把香炉烟线推向走廊方向。门停住后，烛火回弹两次趋于稳定，烟线边缘扩散并在冷空气中逐渐变淡。
```

### 琴弦 + 手部 + 衣袖

```text
右手指尖从琴弦侧面轻拨，琴弦先向内弯曲再快速回弹，细小震动在金色弦线上闪出短促高光。左手按弦时袖口被动作惯性带起一小段，随后因布料重量自然回落，手腕停住后袖口边缘仍有半秒延迟摆动。
```

## 常见错误

- 只有“风很大”，没有任何受力对象。
- 头发、衣服、云雾、雨线、水面朝不同方向运动。
- 所有物体同速漂浮。
- 布料像纸片，没有重量和回落。
- 烟雾没有来源、路径和消散。
- 雨雪没有方向、密度、遮挡和落点。
- 水面无接触点却全屏波动。
- 硬道具无转轴、无摩擦、无停止位置。
- 动作没有物理结果。
- 光线没有和雾、雨、烟、金属、丝绸、水面互动。

## 负面提示词建议

当场景包含复杂动态时，在负面提示词中加入“动力学类”或合并入“动作类 / 风力类 / 场景类”：

```text
动力学类：动力来源不明、物体无原因漂浮、所有物体同速运动、风向与雨线冲突、烟雾无来源无消散、水面无接触点乱波动、布料无重量感、饰品摆动过大、硬道具无支点无摩擦、动作没有物理结果、粒子无落点、光线与雾雨烟尘不互动、跨镜头动态状态跳变
```

如果最终格式不宜新增“动力学类”，可并入：

- 动作类：动作无物理结果、接触无反应、硬道具无支点。
- 风力类：风向混乱、布料无重量、所有物体同速漂浮。
- 场景类：烟雾无来源、水面乱波动、雨雪方向混乱、粒子无消散。

## 与其他规则的关系

- `motion-performance-contract.md`：负责人物、道具、产品动作的动作合约；本规则负责动作造成的物理响应。
- `lighting-intention-contract.md`：负责光源、色温、阴影和材质反射；本规则补充雾、烟、雨、雪、灰尘、水面等介质如何吃光、散射和遮挡。
- `pro-shot-continuity-contract.md`：负责跨镜头连续；本规则补充风向、雨线、水面、烟雾、道具状态的连续性。
- `seedance-wind-softbody-standard.md`：保留为风 / 柔体专项入口；复杂动态以本规则为上位标准。
- `人物镜头强制覆盖规则`：负责触发人物动作、表情、口型、面光、动态材质和参考图一致性；本规则负责动态材质和环境物理的具体写法。
