# Examples Guide

案例用于示范如何理解输入、拆解、构思和生成提示词。遇到相似需求可参考结构，但必须按当前用户需求重新生成，不能照抄。

## V0.6 参考库隔离示例

### 输入：演唱会慢歌，直接生成

系统内部可参考演唱会动作库和镜头库，但最终输出只能写当前项目的可执行镜头：

正确方向：
- 稳定中景承载主唱动作。
- 胸前握麦、闭眼、低头、手按胸口等动作服务抒情。
- 追光、侧光、背景暗部和缓慢推近服务情绪。
- 结尾用拉远或观众灯海收束。

禁止方向：
- 提到参考库来源、原视频、原歌手、原时间戳。
- 复制某个原片具体桥段。
- 在最终输出中出现动作标签名或库名。

### 输入：多场景爱情 MV

正确方向：
- 设定一个主空间作为情绪锚点。
- 每个副空间承载一种关系状态或情绪侧面。
- 主歌镜头较稳，预副歌加快，副歌多空间并置，结尾用统一动作或光影收束。

禁止方向：
- 复刻原始 MV 的具体场景组合。
- 输出可识别的原始人物、道具组合、片名或桥段。
