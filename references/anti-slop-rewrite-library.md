# Anti-Slop Rewrite Library 反空泛词重写库

## 目的

本文件用于把空泛、评价型、堆叠式提示词改写成可拍摄、可看见、可执行的镜头语言。它吸收外部 Seedance anti-slop 思路，但完全以 prompt-script-master 的中文输出结构和参考库隔离规则为准。

## 启动条件

出现以下任一情况时读取本文件：

```text
电影感 / 高级感 / 氛围感 / 质感好 / 震撼 / 唯美 / 梦幻 / 真实感 / 动感 / 大片感 / 更专业 / 更细腻 / 更有故事感 / 提示词太空 / tag堆叠 / 8K / masterpiece / 超高清 / 不要AI感 / 不要塑料 / 不要崩
```

## 可见性测试

每个主要短语都必须通过至少一个测试：

```text
摄像机能看到吗？
灯光师能布出来吗？
演员能演出来吗？
道具或材质能呈现吗？
声音或节奏能听到吗？
时间轴里能发生吗？
```

如果不能，就改写。

## 六类空泛词

| 类型 | 常见形式 | 修复方式 |
|---|---|---|
| 空评价 | 电影感、高级、唯美、震撼 | 转成镜头、光源、构图、材质、动作 |
| 图像模型词 | 8K、masterpiece、best quality、超高清 | 删除；质量不是视频提示词内容 |
| 标签汤 | 女孩, 古风, 美, 电影感, 高质量 | 改成完整拍摄句：主体 + 动作 + 镜头 + 光影 |
| 否定堆叠 | 不要崩、不要糊、不要塑料 | 正文写正向目标，故障进负面 |
| 形容词叠加 | 极美、绝美、梦幻、细腻 | 选择一个具体可见原因 |
| 情绪后缀 | 氛围感、故事感、感性、vibe | 写出遮挡、光、动作停顿、声音和空间压力 |

## 中文替换表

| 空泛词 | 替换方向 |
|---|---|
| 电影感 | 景别 + 运镜 + 主光方向 + 暗部层次 + 色彩倾向 |
| 高级感 | 材质克制、留白构图、受控反射、低饱和色、干净背景 |
| 氛围感 | 雾/烟/雨/逆光/遮挡/声音/空间距离造成的情绪 |
| 唯美 | 色彩、发丝、衣料、柔光、景深、动作节奏 |
| 震撼 | 物理尺度、低角度、广角压迫、群体运动、巨大对比 |
| 动感 | 具体谁移动、速度、方向、路径、终点 |
| 真实感 | 真实光源、材质缺陷、皮肤纹理、接触阴影、自然曝光限制 |
| 梦幻 | 柔化光、浅景深、低对比、漂浮粒子或雾气的来源与消散 |
| 精致 | 可见细节：刺绣、金属边缘、玉石反光、丝绸褶皱、指尖动作 |
| 故事感 | 人物的选择、停顿、未说出口的动作、空间关系变化 |
| 大片感 | 镜头规模、构图压力、光影层级、运动路径、声画节奏 |

## 否定改正向

| 弱写法 | 强写法 |
|---|---|
| 不要 AI 感 | 像真实相机在现场拍下的一帧，有真实光源、自然曝光限制和轻微现场不完美 |
| 不要塑料皮肤 | 皮肤保留自然毛孔、细微肤色差和环境光反射 |
| 不要动作僵硬 | 动作从肩颈、手腕、衣料和视线中连续传递，有起点和终点 |
| 不要风向乱 | 所有发丝、衣料、纱幔、云雾、水面沿同一物理方向响应 |
| 不要脸崩 | 人物参考图锁定脸型比例、五官气质、鼻唇印象和年龄感 |

## 标签汤重写模板

弱写法：

```text
古风少女，唯美，电影感，高级光影，8K，精致，氛围感，漂亮
```

改写：

```text
一名古风少女站在山亭侧廊，右手轻扶衣袖，眼神从地面缓慢抬向镜头。镜头为中近景缓慢推近，右侧低角度暖金侧逆光勾出发丝和肩颈轮廓，脸部暗侧保留眼窝、颧骨和下颌线细节。披帛和袖口被同一方向山风轻轻拉起后回落，背景云雾被夕光照出柔和层次。
```

## 保留有用风格词

不要过度清洗。以下词如果配有可见执行细节，可以保留：

- noir hallway with venetian-blind shadows → 有明确光影和空间。
- 古风仙侠 + 山风 + 云海 + 侧逆光 + 披帛动态 → 有明确风格成因。
- 90年代 CCD 录像质感 + 低动态范围 + 轻微拖影 → 有明确媒介行为。

## 输出方式

当用户要求审查或优化提示词时，可输出：

```text
【空泛词识别】
- ...

【替换方向】
- 镜头：...
- 光影：...
- 动作：...
- 材质：...
- 声音：...

【压缩后提示词】
...
```

如果用户要求直接生成最终提示词，则只内部执行反空泛词重写，不展示审查过程。
