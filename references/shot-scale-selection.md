# Shot Scale Selection 景别选择规则

## 目的

景别必须服务画面任务。不要在远景里要求细微脸部表演，也不要在特写里安排大幅走位、舞蹈和空间调度。

## 景别适配表

### 超远景 / 大远景

适合：建立世界、空间规模、角色孤独感、军阵、山河、舞台全貌、抵达感。  
不适合：锁脸、口型、眼神光、细小手指动作、复杂服装纹样。

写法：

```text
大远景建立山谷与亭台关系，人物只作为画面中心的细小轮廓，动作保持简单可读，重点在空间层次、光源方向和云雾流动。
```

### 远景 / 全身

适合：走位、舞蹈、队形、服装整体、人物与场景关系、动作方向。  
不适合：复杂口型、眼神变化、面部肌肉、手指精细动作。

### 中景 / 半身

适合：人物行为、道具使用、对话、弹琴、拔剑、回头、轻动作。  
这是 Seedance 人物镜头的稳定默认景别。

### 中近景

适合：情绪、口型、眼神、上半身动作、手部与脸部关系。  
需要补足人物面部光影，动作幅度不宜过大。

### 近景 / 特写

适合：脸部情绪、口型、眼神光、泪光、产品局部、手部细节、材质证明。  
不适合：大幅转身、奔跑、舞蹈、多人调度、复杂空间移动。

### 微距 / 特写细节

适合：珠宝、琴弦、布料、雨滴、水面、火焰、机械、食物、产品材质。  
不适合：完整人物身份和大动作。

## 景别冲突规则

- 锁脸 + 大动作冲突时，优先中景/中近景，减少动作幅度。
- 口型 + 环绕冲突时，优先稳定中近景，减少环绕。
- 群像 + 每个人表情冲突时，优先中远景群体队形，重点人物另开近景。
- 大场景 + 细节材质冲突时，拆成建立远景 + 细节特写。
- 首尾帧距离太大时，不靠一个镜头硬转，拆分 clip。

## 输出原则

最终提示词中每个镜头至少应自然包含景别。复杂镜头还应说明这个景别为什么能承载当前任务。
